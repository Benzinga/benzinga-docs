<div id="introduction">
  ## Introducción
</div>

La API de ratios de ganancias proporciona ratios específicamente relacionados con las ganancias y la rentabilidad de la compañía, como:

* ROA (Return on Assets)
* ROE (Return on Equity)
* ROI (Return on Investment)
