<div id="introduction">
  ## Introducción
</div>

La API de Ticker Trends obtiene datos de tendencias para determinados símbolos de ticker basados en métricas de participación de los usuarios. Ayuda a identificar valores que generan un mayor interés mediante el seguimiento de:

* Puntuaciones de tendencia
* Niveles de actividad
* Participación en diferentes intervalos de tiempo
