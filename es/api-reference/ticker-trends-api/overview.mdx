<div id="introduction">
  ## Introducción
</div>

La API de Ticker Trends obtiene datos de tendencias para símbolos de ticker específicos basados en métricas de interacción de los usuarios. Ayuda a identificar valores que están generando un mayor interés al hacer seguimiento de:

* Puntuaciones de tendencia
* Niveles de actividad
* Interacción en diferentes intervalos de tiempo
