<div id="introduction">
  ## Introducción
</div>

La API de ratios de ganancias proporciona ratios específicamente relacionados con las utilidades y la rentabilidad de las empresas, tales como:

* ROA (Return on Assets)
* ROE (Return on Equity)
* ROI (Return on Investment)
