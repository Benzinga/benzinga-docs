<div id="introduction">
  ## Introducción
</div>

La API de fundamentos de la compañía obtiene datos financieros exhaustivos necesarios para un análisis profundo de la empresa. Cubre:

* **Estados financieros**: Balance general, estado de resultados, estado de flujos de efectivo (original, reexpresado, TTM).
* **Perfil de la compañía**: Descripción del negocio, sector, industria, ejecutivos y sede.
* **Informes de ganancias**: Historial de ingresos, beneficio por acción (EPS), EBITDA y utilidad neta.
* **Clase de acciones y valoración**: Detalles de la clase de acciones, alfa/beta y ratios de valoración.
