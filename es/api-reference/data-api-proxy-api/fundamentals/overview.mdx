<div id="introduction">
  ## Introducción
</div>

La API de Company Fundamentals obtiene datos financieros exhaustivos necesarios para un análisis profundo de empresas. Incluye:

* **Estados financieros**: Balance general, estado de resultados, estado de flujo de efectivo (original, reexpresado, TTM).
* **Perfil de la compañía**: Descripción del negocio, sector, industria, ejecutivos y sede.
* **Informes de resultados**: Historial de ingresos, beneficio por acción (EPS), EBITDA y beneficio neto.
* **Clase de acciones y valoración**: Detalles de la clase de acciones, alfa/beta y ratios de valoración.
