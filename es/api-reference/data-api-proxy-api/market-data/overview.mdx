<div id="introduction">
  ## Introducción
</div>

Las API de datos de mercado proporcionan información de mercado completa, incluidas cotizaciones en diferido, valores más activos (market movers) y datos sobre posiciones cortas para valores.

<div id="market-movers">
  ## Market Movers
</div>

La API de Market Movers proporciona datos en tiempo real sobre los valores con mayores subidas, mayores bajadas y los más activamente negociados. Utiliza este endpoint para identificar:

* **Gainers** - Valores con la mayor variación positiva en el precio
* **Losers** - Valores con la mayor variación negativa en el precio
* **Volume Leaders** - Valores con el mayor volumen de negociación

Cada resultado incluye precio, porcentaje de cambio, volumen, capitalización bursátil, sector y datos de free float.

<div id="short-interest-data">
  ## Datos de posición corta
</div>

La API Short Interest proporciona datos sobre la actividad de ventas en corto para valores. Estos datos son valiosos para comprender el sentimiento del mercado e identificar posibles candidatos a short squeeze.

Las métricas clave incluyen:

* **Total Short Interest** - Número de acciones vendidas en corto
* **Days to Cover** - Días estimados necesarios para cubrir las posiciones cortas en función del volumen promedio
* **Short % of Float** - Porcentaje de las acciones en circulación vendidas en corto
* **Month-over-Month Change** - Cambio intermensual en el short interest con respecto al mes anterior
