<div id="introduction">
  ## Introducción
</div>

Las APIs de datos de mercado proporcionan información de mercado integral, incluyendo cotizaciones diferidas, principales variaciones del mercado y datos sobre posiciones cortas en valores.

<div id="market-movers">
  ## Market Movers
</div>

La API de Market Movers proporciona datos en tiempo real sobre los valores con mayores subidas, caídas y los más negociados activamente. Usa este endpoint para identificar:

* **Gainers** - Valores con el mayor cambio positivo de precio
* **Losers** - Valores con el mayor cambio negativo de precio
* **Volume Leaders** - Valores con el mayor volumen de negociación

Cada resultado incluye precio, cambio porcentual, volumen, capitalización de mercado, sector y datos sobre el float.

</div>

<div id="short-interest-data">
  ## Datos de Short Interest
</div>

La API de Short Interest proporciona datos sobre la actividad de ventas en corto de valores. Estos datos son valiosos para comprender el sentimiento del mercado e identificar posibles candidatos a un short squeeze.

Las métricas clave incluyen:

* **Total Short Interest** - Número de acciones vendidas en corto
* **Days to Cover** - Días estimados para cubrir las posiciones cortas en función del volumen promedio
* **Short % of Float** - Porcentaje del float de acciones vendido en corto
* **Month-over-Month Change** - Cambio en el short interest con respecto al mes anterior
