<div id="introduction">
  ## Introducción
</div>

La API de Operation Ratios proporciona métricas que permiten evaluar la eficiencia operativa de una empresa, tales como:

* Rotación de activos
* Rotación de inventario
* Rotación de cuentas por cobrar
