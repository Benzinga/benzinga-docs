<div id="introduction">
  ## Introducción
</div>

La API Bars obtiene datos históricos de barras de precios OHLCV (Open, High, Low, Close, Volume) para valores especificados. Devuelve datos de precios agregados en función de los intervalos especificados y admite:

* Múltiples símbolos de ticker
* Varios rangos de tiempo (fechas absolutas y relativas)
* Intervalos de tiempo personalizables (1m, 5m, 1h, 1d, etc.)
* Filtrado por sesión de negociación (pre-market, regular, after-hours)
