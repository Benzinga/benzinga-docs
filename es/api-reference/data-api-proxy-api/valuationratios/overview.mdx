<div id="introduction">
  ## Introducción
</div>

La API de Valuation Ratios proporciona métricas clave de valoración para evaluar el valor relativo de los valores. Incluye ratios como:

* P/E (Price-to-Earnings)
* P/S (Price-to-Sales)
* P/B (Price-to-Book)
