<div id="introduction">
  ## Introducción
</div>

La API de Valuation Ratios proporciona métricas de valoración clave para evaluar el valor relativo de los títulos. Entre ellas se incluyen ratios como:

* P/E (precio/beneficio)
* P/S (precio/ventas)
* P/B (precio/valor en libros)
