<div id="introduction">
  ## Introducción
</div>

Esta API REST proporciona endpoints para obtener llamadas de conferencia y resúmenes.

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="fetch-all-calls">
  ### [Obtener todas las llamadas](/es/api-reference/delivery_api/calls/fetch-all-calls)
</div>

Obtiene una lista de llamadas con filtros opcionales y paginación.

<div id="fetch-call-by-id">
  ### [Obtener llamada por ID](/es/api-reference/delivery_api/calls/fetch-call-by-id)
</div>

Obtén información detallada sobre una llamada específica mediante su ID.

<div id="fetch-summaries">
  ### [Obtener resúmenes](/es/api-reference/delivery_api/summaries/fetch-summaries)
</div>

Recupera una lista paginada de resúmenes.

<div id="fetch-summary-by-call-id">
  ### [Obtener resumen por ID de llamada](/es/api-reference/delivery_api/summaries/fetch-summary-by-call-id)
</div>

Obtiene los detalles del resumen para un ID de llamada específico.
