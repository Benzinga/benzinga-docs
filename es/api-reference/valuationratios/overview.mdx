<div id="introduction">
  ## Introducción
</div>

La API de ratios de valoración proporciona métricas clave para evaluar el valor relativo de los valores. Entre ellas se incluyen ratios como:

* P/E (Precio-beneficio)
* P/S (Precio-ventas)
* P/B (Precio-valor en libros)
