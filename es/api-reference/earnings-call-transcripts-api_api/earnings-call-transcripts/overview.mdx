<div id="introduction">
  ## Introducción
</div>

La API de transcripciones de conferencias de resultados proporciona acceso a transcripciones en texto completo y archivos de audio de conferencias de resultados corporativos. Incluye:

* Transcripciones completas de las conferencias trimestrales y anuales de resultados
* Sesiones de preguntas y respuestas e identificación de los oradores
* Marcas de tiempo para una navegación detallada
* URLs de archivos de audio descargables
