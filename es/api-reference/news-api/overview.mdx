<div id="introduction">
  ## Introducción
</div>

La API de Newsfeed de Benzinga proporciona noticias financieras en tiempo real, con un filtrado flexible por:

* Tickers, ISINs y CUSIPs
* Canales y temas
* Rangos de fechas y marcas de tiempo de actualización
* Tipos de contenido y nivel de importancia

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-news">
  ### [Obtener noticias](/es/api-reference/news-api/get-news-items)
</div>

Devuelve datos estructurados de noticias.

<div id="get-news-channels">
  ### [Obtener canales de noticias](/es/api-reference/news-api/channels/get-available-news-channels)
</div>

Devuelve una lista de todos los canales de noticias disponibles para filtrar resultados.

<div id="get-removed-news">
  ### [Obtener noticias eliminadas](/es/api-reference/news-api/get-removed-news)
</div>

Devuelve noticias que han sido eliminadas o corregidas.
