<div id="introduction">
  ## Introducción
</div>

La API de NewsQuantified devuelve datos cuantificados de analítica de noticias con puntuaciones de sentimiento, métricas de relevancia e indicadores de impacto en el mercado. Proporciona datos estructurados que analizan los aspectos cualitativos de la cobertura de noticias, incluidos:

* Polaridad del sentimiento
* Recuento de artículos
* Puntuaciones de tendencia
* Métricas específicas por ticker

Estos datos están diseñados para trading algorítmico e investigación cuantitativa.
