<div id="introduction">
  ## Introducción
</div>

La API Bulls Say Bears Say devuelve los últimos casos de inversión alcistas y bajistas para un determinado símbolo de cotización de una acción.

* **Casos alcistas (Bull Cases)**: Presentan argumentos positivos para comprar una acción.
* **Casos bajistas (Bear Cases)**: Presentan argumentos negativos en contra de invertir en una acción.

Estos datos son valiosos para entender los puntos de vista opuestos sobre el rendimiento potencial de un valor.
