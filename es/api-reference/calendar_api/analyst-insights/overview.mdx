<div id="introduction">
  ## Introducción
</div>

La API de Analyst Insights ofrece perspectivas de investigación sobre valores, incluyendo análisis detallados y recomendaciones de analistas financieros. Proporciona acceso a:

* Mejoras y rebajas en las recomendaciones de analistas
* Objetivos de precio
* Notas de investigación detalladas
* Contexto de las calificaciones de consenso

</div>

<div id="get-analyst-insights">
  ### [Obtener Analyst Insights](/es/api-reference/calendar_api/analyst-insights/get-analyst-insights-v1)
</div>
