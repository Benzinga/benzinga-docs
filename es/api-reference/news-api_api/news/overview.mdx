<div id="introduction">
  ## Introducción
</div>

La API de Newsfeed de Benzinga proporciona noticias financieras en tiempo real, que abarcan una amplia gama de tipos de contenido, incluida la funcionalidad &quot;Why Is It Moving&quot; (WIIM). Permite un filtrado flexible por:

* Tickers, ISINs y CUSIPs
* Canales y temas
* Rangos de fechas y marcas de tiempo de actualización
* Tipos de contenido y nivel de importancia

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-news">
  ### [Obtener noticias](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Devuelve datos estructurados de noticias.

<div id="get-news-channels">
  ### [Obtener canales de noticias](/es/api-reference/news-api_api/channels/get-available-news-channels)
</div>

Devuelve una lista de todos los canales de noticias disponibles para filtrar resultados.

<div id="get-removed-news">
  ### [Obtener noticias eliminadas](/es/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

Devuelve noticias que han sido eliminadas o corregidas.

<div id="why-is-it-moving-wiim">
  ### [Why Is It Moving (WIIM)](/es/api-reference/news-api_api/news/get-the-news-items)
</div>

Accede a las explicaciones exclusivas de Benzinga, en una sola frase, sobre movimientos bursátiles válidos. Usa el endpoint estándar de noticias para obtener estos ítems.
