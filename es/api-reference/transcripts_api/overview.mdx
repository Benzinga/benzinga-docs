<div id="introduction">
  ## Introducción
</div>

Esta API REST proporciona endpoints para obtener llamadas de resultados y resúmenes.

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="fetch-all-calls">
  ### [Fetch All Calls](/es/api-reference/transcripts_api/calls/fetch-all-calls)
</div>

Obtén una lista de llamadas con filtros y paginación opcionales.

<div id="fetch-call-by-id">
  ### [Obtener llamada por ID](/es/api-reference/transcripts_api/calls/fetch-call-by-id)
</div>

Obtiene información detallada sobre una llamada específica mediante su ID.

<div id="fetch-summaries">
  ### [Obtener resúmenes](/es/api-reference/transcripts_api/summaries/fetch-summaries)
</div>

Obtén una lista de resúmenes con paginación.

<div id="fetch-summary-by-call-id">
  ### [Obtener resumen por ID de llamada](/es/api-reference/transcripts_api/summaries/fetch-summary-by-call-id)
</div>

Obtiene los detalles del resumen para un ID de llamada específico.
