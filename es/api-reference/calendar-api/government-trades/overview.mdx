<div id="introduction">
  ## Introducción
</div>

La API Government Trades realiza un seguimiento de la actividad de negociación de funcionarios del gobierno de EE. UU., incluidos miembros de la Cámara de Representantes y del Senado. Proporciona transparencia en:

* Informes periódicos de transacciones
* Registros de operaciones individuales
* Tipos de activos y montos de las transacciones

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-government-trades">
  ### [Obtener operaciones gubernamentales](/es/api-reference/calendar_api/government-trades/government-trades)
</div>

Devuelve las operaciones de funcionarios gubernamentales, incluidas las transacciones realizadas por miembros del Congreso.

<div id="get-government-trade-reports">
  ### [Obtener informes de operaciones gubernamentales](/es/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

Devuelve informes detallados de divulgación de operaciones gubernamentales.
