<div id="introduction">
  ## Introducción
</div>

La Government Trades API rastrea la actividad de negociación de funcionarios del gobierno de EE. UU., incluidos miembros de la Cámara de Representantes y del Senado. Proporciona transparencia sobre:

* Informes periódicos de transacciones
* Registros individuales de operaciones
* Tipos de activos y montos de las transacciones

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-government-trades">
  ### [Obtener operaciones de gobierno](/es/api-reference/calendar_api/government-trades/government-trades)
</div>

Devuelve las operaciones de funcionarios del gobierno, incluidas las realizadas por miembros del Congreso.

<div id="get-government-trade-reports">
  ### [Obtener informes de operaciones gubernamentales](/es/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

Devuelve informes detallados sobre la divulgación de operaciones gubernamentales.
