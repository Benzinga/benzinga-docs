<div id="introduction">
  ## Introducción
</div>

La API de Insider Transactions proporciona datos sobre operaciones con acciones realizadas por insiders de la empresa, como directivos, miembros del consejo y accionistas mayoritarios. Incluye datos derivados de declaraciones del Formulario 4 de la SEC.

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-insider-transaction-filing">
  ### [Obtener Insider Transaction Filing](/es/api-reference/calendar-api/insider-transaction/get-insider-transaction-filing)
</div>

Devuelve registros de operaciones de insiders, incluidas compras y ventas de acciones y ejercicios de opciones.

<div id="get-insider-owner">
  ### [Obtener propietario insider](/es/api-reference/calendar-api/insider-transaction/get-insider-transaction-owner)
</div>

Devuelve información sobre propietarios insider.
