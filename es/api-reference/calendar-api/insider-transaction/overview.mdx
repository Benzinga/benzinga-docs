<div id="introduction">
  ## Introducción
</div>

La API de transacciones internas proporciona datos sobre la compraventa de acciones por parte de insiders de la empresa, como directivos, miembros del consejo de administración y accionistas principales. Incluye datos derivados de los formularios Form 4 de la SEC.

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-insider-transaction-filing">
  ### [Obtener presentaciones de operaciones de insiders](/es/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

Devuelve presentaciones de operaciones de insiders, incluidas compras y ventas de acciones, y ejercicios de opciones.

<div id="get-insider-owner">
  ### [Obtener propietario interno](/es/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

Devuelve información sobre los propietarios internos.
