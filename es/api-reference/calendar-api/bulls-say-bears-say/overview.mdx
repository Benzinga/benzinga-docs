<div id="introduction">
  ## Introducción
</div>

La API Bulls Say Bears Say devuelve los casos de inversión alcistas y bajistas más recientes para un símbolo ticker de una acción determinado.

* **Casos alcistas (Bull Cases)**: Presentan argumentos positivos para comprar una acción.
* **Casos bajistas (Bear Cases)**: Presentan argumentos negativos en contra de invertir en una acción.

Estos datos son valiosos para comprender los puntos de vista opuestos sobre las perspectivas de rendimiento de un valor.
