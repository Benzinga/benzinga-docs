<div id="introduction">
  ## Introducción
</div>

La suite de la API Calendars proporciona acceso a una amplia gama de datos de calendario financiero, ayudando a los inversores a hacer seguimiento de eventos corporativos y económicos clave.

<div id="available-groups">
  ## Grupos disponibles
</div>

| Grupo                        | Descripción                                                       | Endpoint                                                   |
| :--------------------------- | :---------------------------------------------------------------- | :--------------------------------------------------------- |
| **Conferencias telefónicas** | Calendario y detalles de las conferencias telefónicas de empresas | [Enlace](/es/api-reference/calendar-api/get-conference-calls) |
| **Dividendos**               | Historial y calendario de pagos de dividendos                     | [Enlace](/es/api-reference/calendar-api/get-dividends)        |
| **Resultados**               | Calendario de presentación de resultados corporativos             | [Enlace](/es/api-reference/calendar-api/get-earnings)         |
| **Economía**                 | Calendario de eventos económicos globales                         | [Enlace](/es/api-reference/calendar-api/get-economics)        |
| **Previsiones**              | Previsiones financieras corporativas                              | [Enlace](/es/api-reference/calendar-api/get-guidance)         |
| **IPOs**                     | Datos de ofertas públicas iniciales                               | [Enlace](/es/api-reference/calendar-api/get-ipos)             |
| **Splits**                   | Historial y calendario de desdoblamientos de acciones             | [Enlace](/es/api-reference/calendar-api/get-splits)           |
| **Calificaciones**           | Calificaciones detalladas de analistas                            | [Enlace](/es/api-reference/calendar-api/get-ratings)          |
