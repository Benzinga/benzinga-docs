<div id="introduction">
  ## Introducción
</div>

La suite de la API Calendars proporciona acceso a una amplia gama de datos de calendario financiero, ayudando a los inversores a hacer seguimiento de eventos corporativos y económicos clave.

<div id="available-groups">

</div>

| Grupo | Descripción | Endpoint |
| :--- | :--- | :--- |
| **Conference Calls** | Programación y detalles de las llamadas de conferencia de la empresa | [Link](/es/api-reference/calendar_api/conference_calls/returns-conference-call-data) |
| **Dividends** | Historial y programación de pagos de dividendos | [Link](/es/api-reference/calendar_api/dividends/dividends-events-v22) |
