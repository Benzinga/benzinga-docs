<div id="introduction">
  ## Introducción
</div>

La suite de la API Calendars proporciona acceso a una amplia gama de datos de calendario financiero, ayudando a los inversores a hacer seguimiento de eventos corporativos y económicos clave.

<div id="available-groups">
  ## Grupos disponibles
</div>

| Grupo | Descripción | Endpoint |
| :--- | :--- | :--- |
| **Conference Calls** | Programación y detalles de las llamadas de conferencia de la empresa | [Link](/es/api-reference/calendar-api/get-conference-calls) |
| **Dividends** | Historial y programación de pagos de dividendos | [Link](/es/api-reference/calendar-api/get-dividends) |
| **Earnings** | Calendario de reportes de resultados corporativos | [Link](/es/api-reference/calendar-api/get-earnings) |
| **Economics** | Calendario de eventos económicos globales | [Link](/es/api-reference/calendar-api/get-economics) |
| **Guidance** | Proyecciones financieras corporativas | [Link](/es/api-reference/calendar-api/get-guidance) |
| **IPOs** | Datos de ofertas públicas iniciales (Initial Public Offerings) | [Link](/es/api-reference/calendar-api/get-ipos) |
| **Splits** | Historial y programación de splits de acciones | [Link](/es/api-reference/calendar-api/get-splits) |
| **Ratings** | Calificaciones detalladas de analistas | [Link](/es/api-reference/calendar-api/get-ratings) |
| **M&amp;A** | Actividad de fusiones y adquisiciones (M&amp;A) | [Link](/es/api-reference/calendar-api/get-ma) |
