<div id="introduction">
  ## Introducción
</div>

La API de Splits proporciona información sobre desdoblamientos de acciones históricos y próximos. Incluye detalles como:

* Relación de desdoblamiento
* Fecha de anuncio
* Fecha ex
* Fecha de pago

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

<div id="get-splits">
  ### [Obtener Splits](/es/api-reference/calendar_api/splits/returns-the-splits-calendar-data)
</div>

Devuelve los datos del calendario de splits.
