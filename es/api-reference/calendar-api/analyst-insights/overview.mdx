<div id="introduction">
  ## Introducción
</div>

La API Analyst Insights devuelve perspectivas de investigación sobre valores, incluidos análisis detallados y recomendaciones de analistas financieros. Proporciona acceso a:

* Revisiones al alza y a la baja de analistas
* Objetivos de precio
* Notas de investigación detalladas
* Contexto de calificaciones de consenso
