<div id="introduction">
  ## Introducción
</div>

Las API de datos de mercado proporcionan información completa del mercado, incluidas cotizaciones diferidas, movimientos del mercado y datos de Short Interest para valores.

<div id="market-movers">
  ## Market Movers
</div>

La API de Market Movers proporciona datos en tiempo real sobre los valores con mayores subidas, mayores caídas y mayor volumen de negociación. Use este endpoint para identificar:

* **Ganadores** - Valores con la mayor variación positiva del precio
* **Perdedores** - Valores con la mayor variación negativa del precio
* **Líderes por volumen** - Valores con el mayor volumen de negociación

Cada resultado incluye el precio, el porcentaje de variación, el volumen, la capitalización de mercado, el sector y datos sobre el capital flotante.

</div>

<div id="short-interest-data">
  ## dato de Short Interest
</div>

La API de Short Interest proporciona datos sobre la actividad de ventas en corto de valores. Estos datos son útiles para comprender el sentimiento del mercado e identificar posibles candidatos a un short squeeze.

Las métricas clave incluyen:

* **Total Short Interest** - Número de acciones vendidas en corto
* **Days to Cover** - Días estimados para cubrir posiciones cortas en función del volumen promedio
* **Short % of Float** - Porcentaje del capital flotante vendido en corto
* **Month-over-Month Change** - Cambio en el Short Interest con respecto al mes anterior
