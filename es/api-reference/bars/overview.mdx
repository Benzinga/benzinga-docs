<div id="introduction">
  ## Introducción
</div>

La API de barra obtiene datos históricos de barras de precios OHLCV (apertura, máximo, mínimo, cierre, volumen) para valores específicos. Devuelve datos de precios agregados según los intervalos definidos y admite:

* Múltiples símbolos bursátiles
* Varios rangos de tiempo (fechas absolutas y relativas)
* Intervalos de tiempo personalizables (1m, 5m, 1h, 1d, etc.)
* Filtrado por sesión de trading (pre-market, regular, after-hours)
