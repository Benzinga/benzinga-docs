<div id="introduction">
  ## Introducción
</div>

La API de ratios de beneficios proporciona ratios específicamente relacionados con los beneficios y la rentabilidad de la empresa, tales como:

* ROA (rendimiento sobre activos)
* ROE (rendimiento sobre el capital)
* ROI (retorno de la inversión)
