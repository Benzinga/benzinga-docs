<div id="introduction">
  ## Introducción
</div>

La Bars API obtiene datos históricos de barras de precios OHLCV (apertura, máximo, mínimo, cierre y volumen) para los valores especificados. Devuelve datos de precios agregados basados en los intervalos indicados y admite:

* Múltiples símbolos de ticker
* Varios rangos de tiempo (fechas absolutas y relativas)
* Intervalos de tiempo personalizables (1m, 5m, 1h, 1d, etc.)
* Filtrado por sesión de negociación (pre-market, regular, after-hours)
