---
title: "Flujo de transcripciones de llamadas de resultados"
description: "Este endpoint WebSocket permite suscribirse a transcripciones de llamadas de resultados en tiempo real."
---

<div id="websocket-websocketv1">
  ## WebSocket `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```


<div id="parameters">
  ### Parámetros:
</div>

| Nombre    | Ubicación |  Tipo   | Obligatorio | Descripción                                                                 |
| :-------- | :-------: | :-----: | ----------: | --------------------------------------------------------------------------: |
| `token`   | query     | string  |        Sí   | token de WebSocket de Benzinga (bz.production***).                          |
| `tickers` | query     | string  |   Opcional  | Lista de símbolos bursátiles a los que deseas suscribirte, separados por comas. |

<div id="stream-response">
  ### Respuesta del stream:
</div>

Cuando te suscribas al stream, recibirás actualizaciones sobre transcripciones de conferencias telefónicas en el siguiente formato:

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"Estamos extremadamente complacidos con nuestros resultados del cuarto trimestre y del año fiscal completo 2024.",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```


<div id="responses">
  #### Respuestas
</div>

- **200 OK**
- **Descripción**: Éxito
- **Tipo de contenido**: `application/json`
- **Esquema**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property      | Type   | Description                                                     |
|---------------|--------|-----------------------------------------------------------------|
| api_version   | string | Versión de la API de WebSocket: `EarningsCallTranscripts/v1`.  |
| kind          | string | Tipo de flujo de conexión de WebSocket.                        |
| data          | object | Contiene detalles sobre los datos del WebSocket.               |
| - type      | string | Tipo de mensaje (por ejemplo, transcript_chunk, transcription_end). |
| - content     | object | Referencia al esquema `EarningsCallTranscripts`.               |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property           | Type    | Description                                                     |
|--------------------|---------|-----------------------------------------------------------------|
| call_id             | string  | ID de llamada único.    |
| transcript_chunk   | string  | Fragmento transcrito de la llamada de resultados.                            |
| security           | object  | Contiene los detalles del valor para el cual se proporciona la información.  |
| - cik              | string  | Central Index Key del valor.                            |
| - exchange         | string  | Exchange (mercado) donde cotiza el valor.                        |
| - isin             | string  | Número Internacional de Identificación de Valores (ISIN).               |
| - name             | string  | Nombre del valor.                                        |
| - symbol           | string  | Símbolo (ticker) del valor.                               |
| time            | integer | Marca de tiempo en que se crea un fragmento de transcripción (en formato float64). |
| index            | integer | Índice del fragmento de transcripción (en formato int). |
