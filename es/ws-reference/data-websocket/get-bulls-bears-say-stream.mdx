<div id="overview">
  ## Descripción general
</div>

Suscríbete para recibir análisis en tiempo real de escenarios alcistas y bajistas para valores. Obtén perspectivas equilibradas sobre escenarios positivos (alcistas) y negativos (bajistas) para acciones.

<div id="key-features">
  ### Funciones clave
</div>

- **Análisis equilibrado**: Reciba tanto escenarios alcistas como bajistas para obtener una visión completa del mercado
- **Actualizaciones en tiempo real**: Reciba notificaciones instantáneas cuando se creen, actualicen o eliminen escenarios
- **Filtrar por valor**: Suscríbase a tickers o ISIN específicos
- **Sentimiento de mercado**: Comprenda tanto las perspectivas optimistas como las pesimistas

<div id="use-cases">
  ### Casos de uso
</div>

- Crear paneles de análisis de sentimiento que muestren ambos lados de la discusión
- Hacer seguimiento de los cambios en el sentimiento alcista/bajista a lo largo del tiempo
- Alertar a los usuarios sobre cambios significativos en la perspectiva del mercado
- Proporcionar análisis de inversión equilibrado a los clientes

<div id="quick-start">
  ## Inicio rápido
</div>

Usa la herramienta de prueba interactiva de WebSocket de arriba para conectarte y probar la transmisión en tiempo real.

<div id="connection-url">
  ### URL de conexión
</div>

```
wss://api.benzinga.com/api/v1/bulls_bears_say/stream?token=YOUR_TOKEN
```


<div id="query-parameters">
  ### Parámetros de consulta
</div>

| Parámetro | Obligatorio | Descripción |
|-----------|-------------|-------------|
| `token` | Sí | Tu token de la API WebSocket de Benzinga (bz.production***) |
| `tickers` | No | Lista separada por comas de símbolos ticker (p. ej., `AAPL,TSLA`) |
| `isins` | No | Lista separada por comas de códigos ISIN para filtrar |

<div id="message-example">
  ## Ejemplo de mensaje
</div>

```json
{
  "id": "websocket123",
  "api_version": "websocket/v1",
  "kind": "bull_bear_case",
  "data": {
    "action": "created",
    "id": "case123",
    "timestamp": "2024-10-01T10:00:00Z",
    "content": {
      "bull_case": "Strong earnings growth and market expansion expected.",
      "bear_case": "Increased competition and regulatory pressures.",
      "ticker": "AAPL",
      "updated": 1696153200
    }
  }
}
```


<div id="message-actions">
  ### Acciones de mensaje
</div>

| Acción | Descripción |
|--------|-------------|
| `created` / `Created` | Nuevo caso alcista o bajista publicado |
| `updated` / `Updated` | Caso existente modificado |
| `deleted` / `Deleted` | Caso eliminado |

<div id="interactive-commands">
  ## Comandos interactivos
</div>

| Comando | Descripción |
|---------|-------------|
| `ping` | Mantiene la conexión activa (responde con `pong`) |
| `replay` | Reproduce hasta los últimos 100 mensajes almacenados en caché |

<div id="best-practices">
  ## Mejores prácticas
</div>

- **Eliminación de duplicados**: Usa el campo `id` para evitar procesar mensajes duplicados
- **Heartbeat**: Envía comandos de `ping` cada 30-60 segundos
- **Presentación equilibrada**: Muestra tanto los casos alcistas como los bajistas para ofrecer un análisis imparcial
- **Seguimiento de actualizaciones**: Supervisa la marca de tiempo `updated` para identificar cambios recientes
