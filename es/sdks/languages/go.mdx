Si estás trabajando con Go, el paquete oficial [Benzinga Go SDK](https://github.com/benzinga) es la forma más sencilla de obtener y manejar secretos para tu aplicación.

* [Paquete](https://pkg.go.dev/github.com/benzinga/go-sdk)
* [Repositorio de GitHub](https://github.com/benzinga/go-sdk)

<div id="basic-usage">
  # Uso básico
</div>

Este ejemplo muestra cómo usar el SDK de Benzinga para Go en una aplicación sencilla de Go. La aplicación obtiene un secreto llamado `API_KEY` del entorno `dev` del proyecto `YOUR_PROJECT_ID`.

<Warning>
  No recomendamos hardcodear tus [tokens de API](/es/platform/identities/overview) en el código. Lo mejor es definirlos como variables de entorno.
</Warning>

<div id="installation">
  # Instalación
</div>

```console
$ go get github.com/benzinga/go-sdk
```
