<div id="creating-a-custom-snippet">
  ## Creación de un fragmento personalizado
</div>

**Requisito previo**: Debe crear su archivo de fragmento en el directorio `snippets`.

<Note>
  Cualquier página del directorio `snippets` se tratará como un fragmento y no
  se renderizará como una página independiente. Si desea crear una página independiente
  a partir del fragmento, impórtelo en otro archivo y úselo como un
  componente.
</Note>

<div id="default-export">
  ### Exportación predeterminada
</div>

1. Agrega contenido a tu archivo de snippet que quieras reutilizar en varias
   ubicaciones. Opcionalmente, puedes agregar variables que se puedan rellenar mediante props
   cuando importes el snippet.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  El contenido que deseas reutilizar debe estar dentro del directorio `snippets`
  para que la importación funcione.
</Warning>

2. Importa el snippet en tu archivo de destino.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```


<div id="reusable-variables">
  ### Variables reutilizables
</div>

1. Exporta una variable desde tu archivo de snippets:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importa el fragmento de código desde tu archivo de destino y utiliza la variable:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hola, mi nombre es {myName} y me gusta {myObject.fruit}.
```


<div id="reusable-components">
  ### Componentes reutilizables
</div>

1. Dentro de tu archivo de snippet, crea un componente que reciba props exportando
   tu componente como una función de flecha.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX no se puede compilar dentro del cuerpo de una función flecha. Procura usar la sintaxis
  HTML siempre que puedas o usa un `default export` si necesitas usar MDX.
</Warning>

2. Importa el snippet en tu archivo de destino y pásale las props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
