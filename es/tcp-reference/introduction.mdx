<div id="key-features">
  ## Características clave
</div>

* **Transmisión en tiempo real**: Reciba noticias financieras y eventos de mercado en cuanto se producen
* **Baja latencia**: La conexión TCP directa minimiza los retrasos en la entrega
* **Formato JSON**: Todos los mensajes se entregan como objetos JSON estructurados
* **Conexión persistente**: Mantenga un flujo continuo de datos de mercado

<div id="prerequisites">
  ## Requisitos previos
</div>

Para conectarte al servicio TCP de Benzinga, necesitas:

| Requisito | Descripción |
|-------------|-------------|
| **Username** | El nombre de usuario de tu cuenta TCP de Benzinga |
| **API Key** | Tu clave de acceso única a la API |

Ponte en contacto con el soporte de Benzinga para obtener tus credenciales TCP.

<div id="available-data">
  ## Datos disponibles
</div>

El flujo TCP entrega diferentes tipos de contenido financiero, entre ellos:

* **Alertas de opciones**: Actividad inusual de opciones, sweeps y operaciones en bloque
* **Comunicados de prensa**: Anuncios corporativos y noticias
* **Suspensiones de negociación**: Notificaciones en tiempo real de suspensión y reanudación
* **Calificaciones de analistas**: Cambios en PO y actualizaciones de calificación
* **Operaciones en bloque**: Grandes transacciones institucionales
