## Introduction

The NewsQuantified API returns quantified news analytics data with sentiment scores, relevance metrics, and market impact indicators. It provides structured data analyzing the qualitative aspects of news coverage, including:
- Sentiment polarity
- Article counts
- Trending scores
- Ticker-specific metrics

This data is designed for algorithmic trading and quantitative research.
