## Introduction

This REST API provides endpoints to fetch conference calls and summaries.

## Available Endpoints

### [Fetch All Calls](/api-reference/transcripts_api/calls/fetch-all-calls)
Retrieve a list of calls with optional filtering and pagination.

### [Fetch Call by ID](/api-reference/transcripts_api/calls/fetch-call-by-id)
Retrieve detailed information about a specific call by its ID.

### [Fetch Summaries](/api-reference/transcripts_api/summaries/fetch-summaries)
Retrieve a list of summaries with pagination.

### [Fetch Summary by Call ID](/api-reference/transcripts_api/summaries/fetch-summary-by-call-id)
Retrieve summary details for a specific call ID.
