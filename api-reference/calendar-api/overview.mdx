## Introduction

The Calendars API suite provides access to a wide range of financial calendar data, helping investors track key corporate and economic events.

## Available Groups

| Group | Description | Endpoint |
| :--- | :--- | :--- |
| **Conference Calls** | Schedule and details of company conference calls | [Link](/api-reference/calendar-api/get-conference-calls) |
| **Dividends** | History and schedule of dividend payments | [Link](/api-reference/calendar-api/get-dividends) |
| **Earnings** | Corporate earnings reports schedule | [Link](/api-reference/calendar-api/get-earnings) |
| **Economics** | Global economic events calendar | [Link](/api-reference/calendar-api/get-economics) |
| **Guidance** | Corporate financial guidance | [Link](/api-reference/calendar-api/get-guidance) |
| **IPOs** | Initial Public Offerings data | [Link](/api-reference/calendar-api/get-ipos) |
| **Splits** | Stock split history and schedule | [Link](/api-reference/calendar-api/get-splits) |
| **Ratings** | Detailed analyst ratings | [Link](/api-reference/calendar-api/get-ratings) |
| **M&A** | Mergers and acquisitions activity | [Link](/api-reference/calendar-api/get-ma) |
