## Introduction

The Operation Ratios API provides metrics to evaluate a company's operational efficiency, such as:
- Asset turnover
- Inventory turnover
- Receivables turnover

## Available Endpoints

### [Get Operation Ratios](/api-reference/operationratios/get-operation-ratios)
Get operation ratios for a list of symbols.
