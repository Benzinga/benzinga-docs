## Introduction

The Earning Ratios API provides ratios specifically related to company earnings and profitability, such as:
- ROA (Return on Assets)
- ROE (Return on Equity)
- ROI (Return on Investment)

## Available Endpoints

### [Get Earning Ratios](/api-reference/earningratios/get-earning-ratios)
Get earning ratios for a list of symbols.
