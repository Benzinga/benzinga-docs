## Introduction

The Earnings Call Transcripts API provides access to full text transcripts and audio files of corporate earnings calls. It includes:
- Full text transcripts of quarterly and annual earnings calls
- Q&A sessions and speaker identification
- Timestamps for granular navigation
- Downloadable audio file URLs

## Available Endpoints

### [Get Earnings Call Transcripts](/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcripts)
Retrieve earnings call transcripts for specified companies or call IDs.

### [Get Earnings Call Audio](/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcript-audio-files)
Retrieve signed URLs for downloading full audio recordings of earnings calls.
