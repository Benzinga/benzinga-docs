## Introduction

The Splits API provides information about historical and upcoming stock splits coverage. It includes details such as:
- Split ratio
- Announcement date
- Ex-date
- Payable date

## Available Endpoints

### [Get Splits](/api-reference/calendar_api/splits/returns-the-splits-calendar-data)
Returns the splits calendar data.
