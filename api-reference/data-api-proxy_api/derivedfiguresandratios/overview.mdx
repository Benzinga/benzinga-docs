## Introduction

The Derived Figures and Ratios API provides calculated financial figures that are derived from fundamental data items. This includes pre-calculated metrics to simplify analysis.

## Available Endpoints

### [Get Derived Figures](/api-reference/data-api-proxy_api/derivedfiguresandratios/get-derived-figures-and-ratios-for-a-list-of-symbols)
Get derived figures and ratios for a list of symbols.
