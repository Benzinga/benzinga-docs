## Introduction

The Quotes Delayed API (similar to Delayed Quotes) provides market quotes with a 15-minute delay.

## Available Endpoints

### [Get Delayed Quotes](/api-reference/data-api-proxy_api/quotedelayed/get-delayed-quotes-for-a-list-of-symbols)
Get delayed quotes for a list of symbols.
