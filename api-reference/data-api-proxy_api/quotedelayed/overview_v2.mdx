## Introduction

The Delayed Quotes API provides market quotes with a 15-minute delay for a list of securities. It supports retrieval by:
- Symbols
- ISINs
- CIKs

### [Get Delayed Quotes v2](/api-reference/data-api-proxy_api/quotedelayed/get-delayed-quotes-v2)
