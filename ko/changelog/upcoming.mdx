<div id="platform-services">
  ### 플랫폼 서비스
</div>

**신규 기능**

- 더 나은 분류를 위한 "Conference Call Topics" 사전 추가
- composite_auto SVG 이미지에 대한 Logos V2 지원
- Halts/Resumes Calendar API용 API 사양 추가
- 이중 반전 로직을 위한 "Hedge Message" 구현
- 모든 서비스 전반의 요청 제한(rate limiting) 및 메시징 개선
- 연결 모니터링을 위한 Corologix 알림 추가

**개선 사항**

- S3 뉴스 전달 모니터링 추가
- 레거시 TDA "Benzinga Attribution" 필드 제거
- Analyst Dictionary에 'Active' 필드 추가(Yes/No 옵션)
- 모든 SoFi 엔드포인트에 대한 ISIN 쿼리 지원
- News 웹소켓 페이로드에 이미지 URL 추가
- 주식 분할과 관련된 JPMC Upside 계산 문제 해결
- Government Trade API 업데이트 처리
- conference call calendar에 기간/연도 데이터 소급 입력(백필)

**버그 수정**

- CIBC 동적 위젯 문제 수정
- 단일 날짜 쿼리에 대해 0을 반환하던 Consensus Ratings 엔드포인트 문제 해결
- /sync 엔드포인트의 중복 제거 로직 수정
- 누락된 Morgan Stanley 우선주 로고 조사

  ### Transcripts
