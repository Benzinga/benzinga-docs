<div id="introduction">
  ## 소개
</div>

Bars API는 지정된 종목에 대한 과거 OHLCV (시가, 고가, 저가, 종가, 거래량) 가격 바(bar) 데이터를 조회합니다. 지정한 간격에 따라 집계된 가격 데이터를 반환하며, 다음을 지원합니다:

* 여러 티커
* 다양한 시간 범위 (절대 및 상대 날짜)
* 사용자 정의 시간 구간 (1m, 5m, 1h, 1d 등)
* 거래 세션 필터링 (장전, 정규장, 장후)
