<div id="introduction">
  ## 소개
</div>

Benzinga Newsfeed API는 &quot;Why Is It Moving&quot;(WIIM) 기능을 포함한 다양한 유형의 콘텐츠를 실시간 금융 뉴스로 제공합니다. 다음과 같은 기준으로 유연하게 필터링할 수 있습니다:

* 티커, 국제증권식별번호(ISIN), CUSIP
* 채널 및 주제
* 날짜 범위 및 업데이트 시각
* 콘텐츠 유형 및 중요도

<div id="why-is-it-moving-wiim">
  ### [Why Is It Moving (WIIM)](/ko/api-reference/news-api_api/news/get-the-news-items)
</div>

Benzinga의 대표 기능인 주가 변동 이유에 대한 한 문장 설명을 조회할 수 있습니다. 이러한 항목을 조회하려면 표준 뉴스 엔드포인트를 사용하십시오.
