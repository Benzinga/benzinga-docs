<div id="introduction">
  ## 소개
</div>

Why Is It Moving (WIIMs)는 &quot;왜 이 종목이 움직이는가?&quot;라는 핵심 질문에 답하기 위한 특화된 뉴스 피드입니다. 선별된 콘텐츠를 통해 의미 있는 주가 변동의 배경과 촉발 요인을 설명하는 맥락 있는 뉴스와 분석을 제공하여, 실시간으로 시장 움직임 뒤에 있는 &quot;이유&quot;를 이해하는 데 도움을 줍니다.

<div id="key-features">
  ## 주요 기능
</div>

* **맥락 기반 뉴스**: 주가 변동의 배경과 원인을 설명하는 데 초점을 맞춘 기사
* **신속한 커버리지**: 즉각적인 주가 움직임을 유발하는 이벤트에 대한 실시간 뉴스
* **실행 가능한 인사이트**: 실적 서프라이즈, 애널리스트 상향/하향 조정, 이례적인 옵션 거래, 긴급 이슈 등 시장에 영향을 미치는 주요 촉매에 대한 분석
* **엄선된 콘텐츠**: 시장에 가장 큰 영향을 주는 이벤트를 선별해 강조한 뉴스

<div id="use-cases">
  ## 사용 사례
</div>

* **시장 움직임 이해**: 예상치 못한 가격 변동의 배경을 신속하게 파악하기
* **트레이딩 인텔리전스**: 트레이딩 결정을 내리기 전에 촉매 요인을 설명하는 뉴스를 사전에 확인하기
* **포트폴리오 모니터링**: 보유 자산에 영향을 미치는 뉴스를 지속적으로 파악하기
* **시장 리서치**: 섹터별로 주가 움직임을 유발하는 요인 패턴을 추적하기
