<div id="introduction">
  ## 소개
</div>

Earnings Call Transcripts API는 기업 실적 발표 콜의 전체 텍스트 기록과 오디오 파일에 대한 접근을 제공합니다. 이 API에는 다음이 포함됩니다:

* 분기 및 연간 실적 발표 콜의 전체 텍스트 기록
* Q&amp;A 세션 및 발표자 식별
* 세밀한 탐색을 위한 타임스탬프
* 다운로드 가능한 오디오 파일 url
