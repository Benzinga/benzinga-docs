<div id="introduction">
  ## 소개
</div>

Insider Transactions API는 임원, 이사, 주요 주주 등 회사 내부자의 주식 거래 데이터를 제공합니다. 여기에는 SEC Form 4 제출 문서에서 추출한 데이터가 포함됩니다.

<div id="available-endpoints">
  ## 제공되는 엔드포인트
</div>

<div id="get-insider-transaction-filing">
  ### [인사이더 거래 공시 가져오기](/ko/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

주식 매수, 매도 및 옵션 행사를 포함한 인사이더 거래 공시를 반환합니다.

<div id="get-insider-owner">
  ### [Insider Owner 가져오기](/ko/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

Insider owner 정보를 반환합니다.
