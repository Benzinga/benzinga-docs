<div id="introduction">
  ## 소개
</div>

Calendars API 제품군은 다양한 금융 일정 데이터에 대한 액세스를 제공하여 투자자가 주요 기업 및 경제 이벤트를 추적할 수 있도록 지원합니다.

<div id="available-groups">
  ## 사용 가능한 그룹
</div>

| Group | Description | Endpoint |
| :--- | :--- | :--- |
| **Conference Calls** | 기업 컨퍼런스콜 일정 및 세부 정보 | [Link](/ko/api-reference/calendar_api/conference_calls/returns-conference-call-data) |
| **Dividends** | 배당금 지급 이력 및 예정 일정 | [Link](/ko/api-reference/calendar_api/dividends/dividends-events-v22) |
| **Earnings** | 기업 실적 발표 일정 | [Link](/ko/api-reference/calendar_api/earnings/returns-the-earnings-data) |
| **Economics** | 글로벌 경제 이벤트 캘린더 | [Link](/ko/api-reference/calendar_api/economics/returns-the-economic-calendar-data) |
| **Guidance** | 기업 실적 가이던스 정보 | [Link](/ko/api-reference/calendar_api/guidance/returns-guidance-data) |
| **IPOs** | 신규 상장(IPO) 데이터 | [Link](/ko/api-reference/calendar_api/ipos/ipos-v21) |
| **Splits** | 주식 분할 이력 및 예정 일정 | [Link](/ko/api-reference/calendar_api/splits/returns-the-splits-calendar-data) |
| **Ratings** | 상세 애널리스트 투자의견 | [Link](/ko/api-reference/calendar_api/ratings/returns-analyst-ratings-data) |
| **M&amp;A** | 기업 인수·합병(M&amp;A) 활동 | [Link](/ko/api-reference/calendar_api/ma/returns-the-mergers-and-acquisitions-data) |
