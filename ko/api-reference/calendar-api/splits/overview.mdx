<div id="introduction">
  ## 소개
</div>

Splits API는 과거 및 예정된 주식 분할 정보를 제공합니다. 다음과 같은 세부 정보를 포함합니다:

* 분할 비율
* 공시일
* 권리락일 (Ex-date)
* 지급일
