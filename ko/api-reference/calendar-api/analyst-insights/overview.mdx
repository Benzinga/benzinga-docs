<div id="introduction">
  ## 소개
</div>

Analyst Insights API는 금융 애널리스트의 상세한 분석과 추천을 포함해, 증권에 대한 리서치 인사이트를 제공합니다. 이를 통해 다음 정보에 접근할 수 있습니다:

* 애널리스트 등급 상향 및 하향
* 목표가
* 상세 리서치 노트
* 컨센서스 레이팅 관련 컨텍스트
