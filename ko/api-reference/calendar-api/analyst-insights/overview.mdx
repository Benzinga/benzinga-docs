<div id="introduction">
  ## 소개
</div>

Analyst Insights API는 금융 애널리스트의 상세한 분석과 추천을 포함한, 개별 증권에 대한 리서치 관점을 제공합니다. 이 API를 통해 다음 데이터를 조회할 수 있습니다:

* 애널리스트 투자의견 상향 및 하향 조정
* 목표가
* 상세 리서치 노트
* 컨센서스 레이팅에 대한 배경 정보
