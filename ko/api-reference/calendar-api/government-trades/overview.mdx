<div id="introduction">
  ## 소개
</div>

Government Trades API는 미국 하원과 상원 의원을 포함한 미국 정부 관계자의 거래 활동을 추적합니다. 이 API는 다음과 같은 항목을 투명하게 공개합니다:

* 정기 거래 보고서
* 개별 거래 기록
* 자산 유형 및 거래 금액

<div id="available-endpoints">
  ## 제공되는 엔드포인트
</div>

<div id="get-government-trades">
  ### [정부 거래 가져오기](/ko/api-reference/calendar_api/government-trades/government-trades)
</div>

정부 공직자(미 의회 의원 포함)의 거래 내역을 반환합니다.

<div id="get-government-trade-reports">
  ### [정부 거래 보고서 가져오기](/ko/api-reference/calendar_api/government-trade-reports/government-trade-reports)
