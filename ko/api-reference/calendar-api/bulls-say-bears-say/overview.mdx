<div id="introduction">
  ## 소개
</div>

Bulls Say Bears Say API는 지정한 주식 티커에 대한 최신 강세 및 약세 투자 논거를 제공합니다.

* **Bull Cases**: 주식을 매수해야 하는 긍정적인 근거를 제시합니다.
* **Bear Cases**: 해당 주식에 투자하지 말아야 하는 부정적인 근거를 제시합니다.

이 데이터는 특정 종목의 향후 성과에 대한 상반된 관점을 이해하는 데 유용합니다.
