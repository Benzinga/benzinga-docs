<div id="introduction">
  ## 소개
</div>

Valuation Ratios API는 종목의 상대적 가치를 평가하는 데 사용되는 핵심 가치평가 지표를 제공합니다. 여기에는 다음과 같은 비율이 포함됩니다:

* P/E (주가수익비율)
* P/S (주가매출비율)
* P/B (주가순자산비율)
