<div id="introduction">
  ## 소개
</div>

수익 비율 API는 기업의 이익 및 수익성과 관련된 비율을 제공합니다. 예를 들면 다음과 같습니다:

* ROA (총자산이익률)
* ROE (자기자본이익률)
* ROI (투자수익률)
