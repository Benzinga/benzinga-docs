<div id="introduction">
  ## 소개
</div>

Valuation Ratios API는 증권의 상대적 가치를 평가하기 위한 핵심 가치평가 지표를 제공합니다. 여기에는 다음과 같은 비율이 포함됩니다:

* P/E (Price-to-Earnings)
* P/S (Price-to-Sales)
* P/B (Price-to-Book)
