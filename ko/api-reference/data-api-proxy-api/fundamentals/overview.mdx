<div id="introduction">
  ## 소개
</div>

Company Fundamentals API는 심층적인 기업 분석에 필요한 포괄적인 재무 데이터를 제공합니다. 이 API는 다음을 포함합니다:

* **재무제표**: 대차대조표, 손익계산서, 현금흐름표(원본, 재작성, TTM).
* **기업 프로필**: 기업 개요, 섹터, 업종, 경영진, 본사 소재지.
* **실적 보고서**: 과거 매출, 주당순이익(EPS), EBITDA 및 순이익.
* **주식 종류 및 가치평가**: 주식 종류 정보, 알파/베타 및 가치평가 비율.
