<div id="introduction">
  ## 소개
</div>

Company Fundamentals API는 심층적인 기업 분석에 필요한 포괄적인 재무 데이터를 제공합니다. 다음 데이터를 다룹니다.

* **재무제표**: 대차대조표, 손익계산서, 현금흐름표(원본, 재작성, TTM 기준).
* **기업 개요**: 사업 설명, 섹터, 업종, 경영진, 본사 소재지.
* **실적 보고서**: 과거 매출, 주당순이익(EPS), 감가상각비·이자·법인세 차감 전 이익(EBITDA), 순이익.
* **주식 종류 및 밸류에이션**: 주식 종류별 세부 정보, 알파/베타, 밸류에이션 비율.
