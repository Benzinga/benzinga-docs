<div id="introduction">
  ## 소개
</div>

Delayed Quotes API는 종목 목록에 대해 15분 지연된 시장 시세를 제공합니다. 다음 방식으로 조회를 지원합니다:

* 심볼
* 국제증권식별번호(ISIN)
* CIK
