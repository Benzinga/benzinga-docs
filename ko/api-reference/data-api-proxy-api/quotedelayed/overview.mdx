<div id="introduction">
  ## 소개
</div>

Delayed Quotes API는 종목 목록에 대해 15분 지연 시세를 제공합니다. 다음 방식으로 조회할 수 있습니다:

* 심볼(symbol)
* 국제증권식별번호(ISIN)
* CIK
