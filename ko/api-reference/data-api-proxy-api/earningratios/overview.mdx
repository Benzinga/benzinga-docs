<div id="introduction">
  ## 소개
</div>

Earning Ratios API는 기업의 실적 및 수익성과 관련된 다음과 같은 지표(비율)를 제공합니다:

* ROA (총자산이익률, Return on Assets)
* ROE (자기자본이익률, Return on Equity)
* ROI (투자수익률, Return on Investment)
