<div id="introduction">
  ## 소개
</div>

Bars API는 지정한 종목에 대한 과거 OHLCV(Open, High, Low, Close, Volume) 가격 바 데이터를 조회합니다. 지정한 간격으로 집계한 가격 데이터를 반환하며, 다음을 지원합니다.

* 여러 티커 심볼
* 다양한 기간(절대 날짜 및 상대 날짜)
* 사용자 정의 시간 버킷(1m, 5m, 1h, 1d 등)
* 거래 세션 필터링(프리마켓, 정규장, 애프터마켓)
