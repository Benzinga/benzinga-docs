<div id="introduction">
  ## 소개
</div>

지연 시세 API는 종목 목록의 시세를 15분 지연되어 제공합니다. 다음 기준으로 조회할 수 있습니다:

* 심볼
* 국제증권식별번호(ISIN)
* CIK
