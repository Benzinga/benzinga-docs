<div id="introduction">
  ## 소개
</div>

Ticker Trends API는 사용자 참여도 지표를 기반으로 지정된 티커 심볼에 대한 트렌드 데이터를 조회합니다. 이는 다음 항목을 추적하여 관심이 증가하고 있는 증권을 식별하는 데 도움이 됩니다:

* 트렌드 점수
* 활동 수준
* 여러 시간 구간에 걸친 참여도
