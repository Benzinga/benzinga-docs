<div id="introduction">
  ## 소개
</div>

이 REST API는 컨퍼런스 콜 및 요약 데이터를 조회하기 위한 엔드포인트를 제공합니다.

<div id="available-endpoints">
  ## 지원되는 엔드포인트
</div>

<div id="fetch-all-calls">
  ### [모든 콜 조회](/ko/api-reference/delivery_api/calls/fetch-all-calls)
</div>

옵션 필터링과 페이지네이션을 통해 콜 목록을 조회합니다.

<div id="fetch-call-by-id">
  ### [ID로 콜 조회](/ko/api-reference/delivery_api/calls/fetch-call-by-id)
</div>

ID로 특정 콜의 상세 정보를 조회합니다.

<div id="fetch-summaries">
  ### [요약 목록 조회](/ko/api-reference/delivery_api/summaries/fetch-summaries)
</div>

페이지네이션이 적용된 요약 목록을 조회합니다.

<div id="fetch-summary-by-call-id">
  ### [Call ID로 요약 가져오기](/ko/api-reference/delivery_api/summaries/fetch-summary-by-call-id)
</div>

특정 Call ID의 요약 정보를 조회합니다.
