<div id="introduction">
  ## 소개
</div>

바(bar) API는 지정된 종목의 과거 OHLCV(시가, 고가, 저가, 종가, 거래량) 바(bar) 데이터를 조회합니다. 지정된 간격을 기준으로 집계된 가격 데이터를 반환하며, 다음을 지원합니다:

* 여러 티커 심볼
* 다양한 기간 범위(절대 날짜 및 상대 날짜)
* 사용자 지정 시간 버킷(1m, 5m, 1h, 1d 등)
* 거래 세션 필터링(장전, 정규장, 장후)
