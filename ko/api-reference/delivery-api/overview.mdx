<div id="introduction">
  ## 소개
</div>

이 REST API는 컨퍼런스 콜 및 요약 정보를 조회할 수 있는 엔드포인트를 제공합니다.

<div id="available-endpoints">
  ## 사용 가능한 엔드포인트
</div>

<div id="fetch-all-calls">
  ### [모든 콜 가져오기](/ko/api-reference/delivery_api/calls/fetch-all-calls)
</div>

선택적 필터링 및 페이징을 통해 콜 목록을 조회합니다.

<div id="fetch-call-by-id">
  ### [ID로 콜 조회](/ko/api-reference/delivery_api/calls/fetch-call-by-id)
</div>

ID로 특정 콜의 상세 정보를 조회합니다.

<div id="fetch-summaries">
  ### [요약 조회](/ko/api-reference/delivery_api/summaries/fetch-summaries)
</div>

페이지네이션을 사용하여 요약 목록을 조회합니다.

<div id="fetch-summary-by-call-id">
  ### [Call ID로 요약 조회](/ko/api-reference/delivery_api/summaries/fetch-summary-by-call-id)
</div>

특정 Call ID에 대한 요약 상세 정보를 조회합니다.
