<div id="introduction">
  ## 소개
</div>

NewsQuantified API는 감성 점수, 연관성 지표, 시장 영향 지표가 포함된 정량화된 뉴스 분석 데이터를 반환합니다. 이 API는 다음을 포함해 뉴스 보도의 정성적 특성을 분석한 구조화된 데이터를 제공합니다:

* 감성 극성
* 기사 수
* 트렌딩 점수
* 티커별 지표

이 데이터는 알고리즘 트레이딩 및 정량적 리서치를 위해 설계되었습니다.
