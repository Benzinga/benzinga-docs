<div id="introduction">
  ## 소개
</div>

NewsQuantified API는 감성 점수, 관련성 지표, 시장 영향 지표를 포함한 정량화된 뉴스 분석 데이터를 반환합니다. 이 API는 뉴스 보도의 정성적 측면을 분석하는 구조화된 데이터를 제공하며, 다음을 포함합니다:

* 감성 극성
* 기사 수
* 트렌드 점수
* 티커별 지표

이 데이터는 알고리즘 트레이딩과 정량적 연구를 위해 설계되었습니다.
