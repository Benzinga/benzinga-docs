<div id="introduction">
  ## 소개
</div>

Benzinga Newsfeed API는 다음과 같은 기준으로 유연하게 필터링할 수 있는 실시간 금융 뉴스를 제공합니다:

* 티커, 국제증권식별번호(ISIN), CUSIP
* 채널 및 주제
* 날짜 범위 및 업데이트 시각
* 콘텐츠 유형 및 중요도
