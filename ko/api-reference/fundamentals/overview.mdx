<div id="introduction">
  ## 소개
</div>

기업 기초 재무 지표 API는 심층적인 기업 분석에 필요한 종합적인 재무 데이터를 제공합니다. 다음 항목을 포함합니다:

* **재무제표**: 대차대조표, 손익계산서, 현금흐름표(원본, 수정본, TTM)
* **기업 프로필**: 사업 설명, 섹터, 산업, 경영진, 본사 위치
* **실적 보고서**: 과거 매출, 주당순이익(EPS), EBITDA, 순이익
* **주식 종류 및 가치평가**: 주식 종류 세부 정보, 알파/베타, 가치평가 비율
