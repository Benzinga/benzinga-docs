  ## 이미지
</div>

<div id="using-markdown">
  ### 마크다운 사용하기
</div>

[마크다운 문법](https://www.markdownguide.org/basic-syntax/#images)을 사용하면 다음 코드를 사용해 이미지를 추가할 수 있습니다.

```md
![title](/path/image.jpg)
```

이미지 파일 크기는 5MB 미만이어야 합니다. 그렇지 않은 경우 [Cloudinary](https://cloudinary.com/) 또는 [S3](https://aws.amazon.com/s3/)와 같은 서비스에 이미지를 업로드할 것을 권장합니다. 그런 다음 해당 URL을 사용해 임베드하면 됩니다.

<div id="using-embeds">
  ### 임베드 사용하기
</div>

이미지를 보다 유연하게 커스터마이즈하려면 [embeds](/ko/writing-content/embed)를 사용해 이미지를 추가할 수도 있습니다

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

문서 내에 다른 HTML 페이지를 불러와 표시합니다. 주로 동영상을 삽입하는 데 사용됩니다.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
