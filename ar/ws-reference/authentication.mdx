---
title: "المصادقة"
description: "كيفية مصادقة الطلبات إلى واجهة برمجة التطبيقات API من Benzinga."
---

للوصول إلى واجهة برمجة التطبيقات API الخاصة بـ Benzinga، تحتاج إلى مصادقة كل طلب باستخدام رمز وصول لـ API. يمكنك إنشاء رموز الوصول لـ API وإدارتها من خلال [Benzinga Console](https://www.api.benzinga.com/token).

<Note>
رمز الوصول لـ API يمنحك صلاحية الوصول إلى حسابك في Benzinga، لذا تعامل معه بشكل آمن. تجنّب مشاركته في المستودعات العامة، أو في الشيفرة العاملة على جانب العميل، أو في أي مواقع أخرى غير آمنة.
</Note>

<div id="authenticating-requests">
  ## مصادقة الطلبات
</div>

تستخدم واجهة برمجة التطبيقات API من Benzinga **مصادقة Bearer**، وهي آلية آمنة يُرفَق فيها بكل طلب رمز وصول في معلمة الاستعلام `token`. يعمل هذا الرمز كحامل لامتيازات الوصول إلى موارد Benzinga.

<div id="example-request-with-bearer-token">
  ### مثال على طلب باستخدام Bearer Token
</div>

ضمِّن رمز الوصول في معامل الاستعلام `Token` ضمن طلبك كما يلي:

```bash
Token: <token>
```
