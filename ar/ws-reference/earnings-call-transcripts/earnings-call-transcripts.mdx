description: "يتيح لك منفذ ويب سوكيت هذا الاشتراك في نصوص مكالمات المؤتمرات الهاتفية في الوقت الفعلي."
---

<div id="websocket-websocketv1">
  ## ويب سوكيت `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```


<div id="parameters">
  ### المعلمات:
</div>

| Name       | Location |  Type   | Required | Description                                                                 |
| :--------- | :-----:  | :-----: | -------: | --------------------------------------------------------------------------: |
| `token`    | query    | string  |      Yes | رمز الوصول الخاص ببث Benzinga عبر ويب سوكيت (bz.production***).              |
| `tickers`  | query    | string  | Optional | قائمة برموز التداول التي تريد الاشتراك فيها، مفصولة بفاصلة.                  |

<div id="stream-response">
  ### استجابة البث:
</div>

عند الاشتراك في البث، ستتلقى تحديثات حول نصوص مكالمات الأرباح (conference calls) بالصيغة التالية:

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"نحن سعداء للغاية بنتائج الربع الرابع والسنة المالية الكاملة 2024.",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```


<div id="responses">
  #### الاستجابات
</div>

- **200 OK**
- **الوصف**: تم بنجاح
- **نوع المحتوى**: `application/json`
- **المخطط**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property      | Type   | Description                                                     |
|---------------|--------|-----------------------------------------------------------------|
| api_version   | string | إصدار واجهة برمجة التطبيقات عبر ويب سوكيت: `EarningsCallTranscripts/v1`.            |
| kind          | string | نوع تدفق اتصال ويب سوكيت.                              |
| data          | object | يحتوي على التفاصيل المتعلقة ببيانات ويب سوكيت.                    |
| - type      | string | نوع الرسالة (مثلًا transcript_chunk, transcription_end).      |
| - content     | object | إشارة إلى مخطط `EarningsCallTranscripts`.                         |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property           | Type    | Description                                                     |
|--------------------|---------|-----------------------------------------------------------------|
| call_id             | string  | معرّف فريد للمكالمة.    |
| transcript_chunk   | string  | جزء من نص مكالمة الأرباح.                            |
| security           | object  | يحتوي على تفاصيل الأداة المالية التي يتم توفير الرؤية التحليلية لها.  |
| - cik              | string  | مفتاح الفهرس المركزي CIK للأداة المالية.                            |
| - exchange         | string  | البورصة التي تُدرج فيها الأداة المالية.                        |
| - isin             | string  | رقم التعريف الدولي للأوراق المالية ISIN.               |
| - name             | string  | اسم الأداة المالية.                                        |
| - symbol           | string  | رمز التداول الخاص بالأداة المالية.                               |
| time            | integer | الطابع الزمني عند إنشاء جزء من نص المكالمة (منسَّق كـ float64). |
| index            | integer | فهرس جزء النص (منسَّق كـ int). |
