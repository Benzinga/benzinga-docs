* استرجاع الأسرار عند الطلب

  ## الأسئلة الشائعة
</div>

<AccordionGroup>
  <Accordion title="هل من المجدي طلب البيانات في كل استدعاء لواجهة برمجة التطبيقات API؟">
    تم تحسين عدة تطوير البرمجيات SDK الخاصة بـ Benzinga لتحقيق أداء عالٍ. فهي تُخزِّن البيانات التي يتم الوصول إليها بشكل متكرر مؤقتًا وتنفّذ استراتيجيات لتقليل الطلبات المكررة، مما يضمن تشغيل تطبيقك بكفاءة. قد تختلف تفاصيل التخزين المؤقت حسب تنفيذ الـ SDK المحدد ولغة البرمجة المستخدمة.
  </Accordion>

  <Accordion title="ماذا يحدث إذا فشل طلب إلى واجهة برمجة التطبيقات API الخاصة بـ Benzinga؟">
    تتضمن الـ SDK آليات لمعالجة الأخطاء وإعادة المحاولة. إذا فشل الطلب، تحاول الـ SDK إعادة المحاولة وفق الإعدادات المعرَّفة أو تُرجع رسائل خطأ مناسبة، مما يمكّن تطبيقك من التعامل مع حالات الفشل بسلاسة.
  </Accordion>

  <Accordion title="هل يمكنني تخصيص إعدادات الـ SDK لتناسب احتياجات تطبيقي؟">
    نعم. تتيح لك عدة تطوير البرمجيات SDK الخاصة بـ Benzinga تخصيص الإعدادات عبر معاملات مثل إعداد مفتاح واجهة برمجة التطبيقات API، وإعدادات مهلة الانتظار، وخيارات أخرى خاصة بحالة الاستخدام لديك. راجع توثيق الـ SDK للغة البرمجة التي تستخدمها لمعرفة المزيد حول المعاملات القابلة للتكوين.
  </Accordion>
</AccordionGroup>
