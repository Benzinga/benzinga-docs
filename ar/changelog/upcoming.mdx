في ما يلي الإصدارات القادمة وعناصر العمل المخطَّط لها في التحديثات المقبلة.

<div id="in-progress">
  ## قيد التنفيذ
</div>

العناصر الجاري تطويرها أو مراجعتها حاليًا.

<div id="in-development">
  ### قيد التطوير
</div>

- **Platform Services**: التحقيق في صفقات الخيارات ذات التأثير العالي التي لا تظهر في استجابات واجهة برمجة التطبيقات API
- **Platform Services**: حل مشكلات مصادقة 401 لنقاط نهاية Halt/Resume
- **Platform Services**: إضافة دعم لجميع أنواع webhook (حالياً يدعم الأخبار فقط)
- **Platform Services**: معالجة واستكمال بيانات محتوى Fastly
- **Platform Services**: أتمتة توليد الصفوف المستقبلية لـ Guidance Calendar
- **Platform Services**: التحقيق في أسلوب التعامل مع التصنيفات للشركات غير النشطة
- **Calendar API**: إضافة حقول الفترة والسنة إلى تقويم Conference Call
- **Calendar API**: تبسيط معلمات واجهة برمجة التطبيقات API عبر جميع خدمات calendar
- **Transcripts**: إضافة حقول الفترة والسنة إلى استدعاءات transcript
- **Transcripts**: تحسين دعم البحث حسب رمز التداول (مثل تنسيق MOG/A)
- **Transcripts**: إضافة ميزة Change Titles لتضمين الفترة والسنة

<div id="in-review">
  ### قيد المراجعة
</div>

- **Platform Services**: تحديث تصنيفات نوع المحتوى «Press Release»

<div id="pending-deployment">

<div id="coming-soon">
  ## قريبًا
</div>

عناصر مخطط لها في السبرنتات القادمة.

<div id="platform-services">
  ### خدمات المنصة
</div>

**ميزات جديدة**

- إضافة قاموس "Conference Call Topics" لتحسين التصنيف
- دعم Logos V2 لصور composite_auto بصيغة SVG
- مواصفات واجهة برمجة التطبيقات API لتقويم Halts/Resumes
- تنفيذ "Hedge Message" لمنطق الانعكاس المزدوج
- تحسينات في تقييد المعدل (Rate limiting) والرسائل عبر جميع الخدمات
- إضافة تنبيهات Corologix لمراقبة الاتصالات

**تحسينات**

- إضافة مراقبة لتسليم الأخبار عبر S3
- إزالة حقل TDA القديم "Benzinga Attribution"
- إضافة حقل 'Active' إلى قاموس المحللين Analyst Dictionary (خيارات Yes/No)
- دعم استعلام رقم التعريف الدولي للأوراق المالية ISIN لجميع نقاط نهاية SoFi
- إضافة رابط صورة إلى حمولة أخبار ويب سوكيت News WebSocket
- حل مشكلات احتساب هامش الصعود JPMC Upside مع تجزئة الأسهم
- التعامل مع تحديثات Government Trade API
- استكمال بيانات الفترة/السنة في تقويم مكالمات المؤتمر (conference call calendar)

**إصلاحات أخطاء**

- إصلاح مشكلات الأداة الديناميكية CIBC dynamic widget
- حل مشكلة نقطة نهاية Consensus Ratings التي تُرجع أصفارًا لاستعلامات التاريخ الفردي
- إصلاح منطق إزالة التكرار في نقطة النهاية /sync
- التحقيق في الشعارات المفقودة للأسهم الممتازة الخاصة بـ Morgan Stanley

<div id="calendar-api">
  ### واجهة برمجة التطبيقات calendar
</div>

**ميزات جديدة**

- إنشاء نقطة نهاية في واجهة برمجة التطبيقات لتحديث Government Trades من أدوات Newsdesk

<div id="news-api">
  ### واجهة برمجة تطبيقات الأخبار (News API)
</div>

**ميزات جديدة**

- دعم أنواع محتوى متعددة في استعلام واحد
- إضافة معلمة استعلام مستندة إلى القطاع
- إضافة دعم لاستعلام مستند إلى GICS

**تصحيحات للأخطاء**

- إضافة دعم لاستعلام updatedSince إلى نقطة النهاية news-removed

<div id="transcripts">
  ### نصوص المكالمات
</div>

**تحسينات**

- عرض جميع النصوص المجدولة للتواريخ المستقبلية
- إضافة دعم عنوان بريد إلكتروني مخصص لمكالمات المؤتمرات
- إضافة دعم رابط مكالمة Zoom
- تحسين واجهة المستخدم مع خيارات تصفية محسَّنة
- إعادة تصميم واجهة المستخدم إلى عرض شبكة التقويم (بدلاً من الوضع اليدوي/الحي)
- إضافة سجل تاريخ المراجعات لكل سجل نص
- إضافة ميزة إنشاء سجل نص يدويًا

**تصحيحات الأخطاء**

- تحسين معالجة تسجيل الدخول في Edge Media
- تعزيز موثوقية وكيل Webvoyager
