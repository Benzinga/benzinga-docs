<div id="introduction">
  ## المقدّمة
</div>

توفّر واجهة برمجة التطبيقات API لتغذية أخبار Benzinga أخبارًا مالية آنية مع إمكانية التصفية المرنة وفقًا لـ:

* رموز التداول، وأرقام التعريف الدولي للأوراق المالية ISIN، و CUSIP
* القنوات والمواضيع
* نطاقات التواريخ والطوابع الزمنية لآخر تحديث
* أنواع المحتوى ومستوى الأهمية

<div id="available-endpoints">
  ## نقاط النهاية المتاحة
</div>

  ### [جلب الأخبار](/ar/api-reference/news-api_api/news/get-the-news-items)

<div id="get-news-channels">
