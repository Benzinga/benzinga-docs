<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات API لتغذية أخبار Benzinga أخبارًا مالية فورية مع إمكانية التصفية المرنة حسب:

* رموز التداول، وأرقام التعريف الدولية للأوراق المالية ISIN، وأكواد CUSIP
* القنوات والمواضيع
* نطاقات التاريخ والطوابع الزمنية لوقت التحديث
* أنواع المحتوى ومستويات الأهمية

<div id="available-endpoints">
  ## نقاط النهاية المتاحة
