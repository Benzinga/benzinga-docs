<div id="introduction">
  ## المقدمة
</div>

واجهة برمجة التطبيقات API لـ Ticker Trends تسترجع بيانات الاتجاهات لرموز التداول المحددة استنادًا إلى مقاييس تفاعل المستخدمين. تساعد في تحديد الأوراق المالية التي تسجّل اهتمامًا متزايدًا من خلال تتبّع:

* درجات الاتجاه
* مستويات النشاط
* التفاعل عبر فترات زمنية مختلفة
