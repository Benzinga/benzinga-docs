<div id="introduction">
  ## المقدمة
</div>

تقوم واجهة برمجة التطبيقات Ticker Trends باسترجاع بيانات الاتجاهات لرموز التداول المحددة استنادًا إلى مقاييس تفاعل المستخدم. تساعد على تحديد الأوراق المالية التي تشهد اهتمامًا متزايدًا من خلال تتبّع:

* درجات الاتجاه
* مستويات النشاط
* التفاعل عبر فترات زمنية مختلفة
