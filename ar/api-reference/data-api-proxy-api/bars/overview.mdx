<div id="introduction">
  ## المقدمة
</div>

تسترجع واجهة برمجة التطبيقات Bars API بيانات الشموع السعرية التاريخية OHLCV (سعر الافتتاح، الأعلى، الأدنى، الإغلاق، الحجم) للأوراق المالية المحددة. تُرجِع بيانات أسعار مُجمَّعة استنادًا إلى الفواصل الزمنية المحددة وتدعم:

* رموز تداول متعددة
* نطاقات زمنية متنوعة (تواريخ مطلقة ونسبية)
* فترات تجميع زمنية مخصصة (1m، 5m، 1h، 1d، إلخ)
* تصفية جلسات التداول (ما قبل الافتتاح، الجلسة العادية، ما بعد الإغلاق)
