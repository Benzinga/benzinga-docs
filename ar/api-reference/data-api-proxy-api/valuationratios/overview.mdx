<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات Valuation Ratios مؤشرات تقييم رئيسية لقياس القيمة النسبية للأوراق المالية. تشمل هذه المؤشرات نسبًا مثل:

* نسبة السعر إلى الأرباح (P/E)
* نسبة السعر إلى المبيعات (P/S)
* نسبة السعر إلى القيمة الدفترية (P/B)
