<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات API الخاصة بنِسَب التقييم مقاييس تقييم رئيسية لتقدير القيمة النسبية للأوراق المالية. تتضمن هذه المقاييس نِسَبًا مثل:

* P/E (السعر إلى الأرباح)
* P/S (السعر إلى المبيعات)
* P/B (السعر إلى القيمة الدفترية)
