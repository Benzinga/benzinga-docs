<div id="introduction">
  ## المقدمة
</div>

تسترجع واجهة برمجة التطبيقات للأساسيات المالية للشركات بيانات مالية شاملة وأساسية لتحليل الشركات بعمق. وهي تغطي:

* **القوائم المالية**: الميزانية العمومية، قائمة الدخل، قائمة التدفقات النقدية (الأصلية، المعاد عرضها، وآخر 12 شهرًا TTM).
* **الملف التعريفي للشركة**: وصف الأعمال، القطاع، الصناعة، المدراء التنفيذيون، والمقر الرئيسي.
* **تقارير الأرباح**: الإيرادات التاريخية، ربحية السهم (EPS)، الأرباح قبل الفوائد والضرائب والإهلاك والاستهلاك (EBITDA)، وصافي الدخل.
* **فئات الأسهم والتقييم**: تفاصيل فئات الأسهم، ألفا/بيتا، ونسب التقييم.
