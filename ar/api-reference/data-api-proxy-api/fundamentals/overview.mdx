<div id="introduction">
  ## المقدمة
</div>

توفّر واجهة برمجة التطبيقات للأساسيات المالية بيانات مالية شاملة ضرورية لإجراء تحليل عميق للشركات. وهي تغطي:

* **القوائم المالية**: الميزانية العمومية، قائمة الدخل، قائمة التدفقات النقدية (أصلية، معاد بيانها، وآخر 12 شهرًا TTM).
* **الملف التعريفي للشركة**: وصف النشاط، القطاع، الصناعة، الإداريون، والمقر الرئيسي.
* **تقارير الأرباح**: الإيرادات التاريخية، ربحية السهم (EPS)، الأرباح قبل الفوائد والضرائب والإهلاك والاستهلاك (EBITDA)، وصافي الدخل.
* **فئة الأسهم والتقييم**: تفاصيل فئة الأسهم، ألفا/بيتا، ونِسَب التقييم.
