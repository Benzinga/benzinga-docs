  ## المقدّمة
</div>

تقدّم واجهة برمجة التطبيقات API لأسعار السوق المتأخرة بيانات أسعار السوق مع تأخير لمدة 15 دقيقة لقائمة من الأوراق المالية. وهي تدعم الاسترجاع حسب:

* الرموز
* أرقام التعريف الدولي للأوراق المالية ISIN
* معرّفات CIK
