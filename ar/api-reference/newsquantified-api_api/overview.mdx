<div id="introduction">
  ## المقدمة
</div>

تُرجِع واجهة برمجة التطبيقات API الخاصة بـ NewsQuantified بيانات تحليلات إخبارية كمية تتضمن درجات المشاعر، ومقاييس الصلة، ومؤشرات تأثير السوق. وهي توفّر بيانات منظَّمة لتحليل الجوانب النوعية للتغطية الإخبارية، بما في ذلك:

* قطبية المشاعر
* أعداد المقالات
* درجات الاتجاه
* مقاييس خاصة برمز التداول

تم تصميم هذه البيانات لدعم التداول الخوارزمي والأبحاث الكمية.
