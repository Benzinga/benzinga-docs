<div id="introduction">
  ## المقدمة
</div>

توفّر واجهة برمجة التطبيقات API لنسبة التقييم مقاييس تقييم أساسية لتقدير القيمة النسبية للأوراق المالية. ومن هذه النسب:

* P/E (السعر إلى الأرباح)
* P/S (السعر إلى المبيعات)
* P/B (السعر إلى القيمة الدفترية)
