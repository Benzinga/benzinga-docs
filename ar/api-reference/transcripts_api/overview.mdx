<div id="available-endpoints">
  ## نقاط النهاية المتاحة
</div>

<div id="fetch-all-calls">
  ### [جلب جميع المكالمات](/ar/api-reference/transcripts_api/calls/fetch-all-calls)

استرداد قائمة بالمكالمات مع إمكانية التصفية وترقيم الصفحات (pagination) بشكل اختياري.

<div id="fetch-call-by-id">

<div id="fetch-summaries">
  ### [جلب الملخصات](/ar/api-reference/transcripts_api/summaries/fetch-summaries)

جلب قائمة من الملخصات مع دعم ترقيم الصفحات.

<div id="fetch-summary-by-call-id">
