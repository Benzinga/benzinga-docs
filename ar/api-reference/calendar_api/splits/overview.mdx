<div id="introduction">
  ## المقدمة
</div>

توفّر واجهة برمجة التطبيقات Splits (Splits API) معلومات حول عمليات تجزئة الأسهم التاريخية والقادمة. وتتضمن تفاصيل مثل:

* نسبة التجزئة
* تاريخ الإعلان
* تاريخ الاستحقاق (Ex-date)
* تاريخ الدفع
