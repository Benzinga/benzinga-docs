<div id="introduction">
  ## المقدمة
</div>

تتيح واجهة برمجة التطبيقات API لأساسيات الشركة المالية جلب بيانات مالية شاملة لازمة لإجراء تحليل متعمّق للشركة. وتشمل:

* **البيانات المالية**: الميزانية العمومية، وقائمة الدخل، وقائمة التدفقات النقدية (الأصلية، والمعدّلة، وآخر اثني عشر شهرًا).
* **الملف التعريفي للشركة**: وصف النشاط، والقطاع، والصناعة، والتنفيذيون، والمقر الرئيسي.
* **تقارير الأرباح**: الإيرادات التاريخية، وربحية السهم (EPS)، وEBITDA، وصافي الدخل.
* **فئة السهم والتقييم**: تفاصيل فئة السهم، وalpha/beta، ونِسَب التقييم.
