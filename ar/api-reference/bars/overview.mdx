<div id="introduction">
  ## المقدمة
</div>

تجلب واجهة برمجة التطبيقات API الخاصة بـ Bars بيانات أشرطة الأسعار التاريخية OHLCV ‏(الافتتاح، الأعلى، الأدنى، الإغلاق، الحجم) للأوراق المالية المحددة. وتُرجِع بيانات أسعار مجمّعة استنادًا إلى الفواصل الزمنية المحددة، كما تدعم:

* عدة رموز أسهم
* نطاقات زمنية متنوعة (تواريخ مطلقة ونسبية)
* تقسيمات زمنية قابلة للتخصيص (1m، 5m، 1h، 1d، إلخ)
* التصفية حسب جلسة التداول (ما قبل السوق، العادية، ما بعد الإغلاق)
