<div id="introduction">
  ## المقدّمة
</div>

توفر واجهة برمجة التطبيقات API للبيانات الصحفية (Press Releases) إمكانية الوصول إلى البيانات الصحفية الرسمية للشركات والإعلانات المؤسسية. تقدّم نقطة النهاية المخصّصة هذه اتصالات رسمية موثوقة للشركات وفي الوقت المناسب، مباشرةً من المصدر، مما يمكّنك من متابعة الأخبار الرسمية للشركات وآخر التحديثات الاستراتيجية.

<div id="key-features">
  ## المزايا الأساسية
</div>

* **إعلانات رسمية**: وصول مباشر إلى البيانات الصحفية الموثوقة للشركات
* **توزيع لحظي**: تسليم فوري للبيانات الصحفية الجديدة عند إصدارها
* **تغطية شاملة**: بيانات صحفية صادرة عن شركات بمختلف فئات القيمة السوقية
* **بيانات منظَّمة**: تنسيق متّسق يسهل التكامل والتحليل

<div id="use-cases">
  ## حالات الاستخدام
</div>

* **مراقبة الامتثال**: تتبّع الإفصاحات الرسمية للشركات والتقارير التنظيمية
* **أبحاث الاستثمار**: الوصول إلى المصادر الأولية لأخبار الشركات وإعلاناتها الرسمية
* **تجميع الأخبار**: بناء منصات أخبار شاملة تستند إلى البيانات الصحفية الرسمية للشركات
* **أنظمة التنبيه**: إنشاء تنبيهات لبيانات صحفية محددة تصدرها الشركات
