<div id="introduction">
  ## المقدمة
</div>

توفر مجموعة واجهات برمجة التطبيقات Calendars إمكانية الوصول إلى مجموعة واسعة من بيانات التقويم المالي، مما يساعد المستثمرين على تتبع الأحداث الرئيسة للشركات والاقتصاد.

<div id="available-groups">
  ## المجموعات المتاحة
</div>

| المجموعة | الوصف | نقطة النهاية |
| :--- | :--- | :--- |
| **المؤتمرات الهاتفية** | جدول وتفاصيل المؤتمرات الهاتفية للشركات | [رابط](/ar/api-reference/calendar-api/get-conference-calls) |
| **توزيعات الأرباح** | سجل وجدول مدفوعات توزيعات الأرباح | [رابط](/ar/api-reference/calendar-api/get-dividends) |
| **الأرباح** | جدول تقارير أرباح الشركات | [رابط](/ar/api-reference/calendar-api/get-earnings) |
| **الاقتصاد** | تقويم الأحداث الاقتصادية العالمية | [رابط](/ar/api-reference/calendar-api/get-economics) |
| **التوجيهات** | التوجيهات المالية للشركات | [رابط](/ar/api-reference/calendar-api/get-guidance) |
| **الاكتتابات العامة الأولية** | بيانات الاكتتابات العامة الأولية | [رابط](/ar/api-reference/calendar-api/get-ipos) |
| **تجزئة الأسهم** | سجل وجدول تجزئة الأسهم | [رابط](/ar/api-reference/calendar-api/get-splits) |
| **التقييمات** | تقييمات المحللين التفصيلية | [رابط](/ar/api-reference/calendar-api/get-ratings) |
| **M&amp;A** | نشاط الاندماج والاستحواذ | [رابط](/ar/api-reference/calendar-api/get-ma) |
