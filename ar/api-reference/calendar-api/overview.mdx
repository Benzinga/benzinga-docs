<div id="introduction">
  ## المقدمة
</div>

توفر مجموعة واجهات برمجة التطبيقات Calendars إمكانية الوصول إلى مجموعة واسعة من بيانات التقويم المالي، مما يساعد المستثمرين على تتبع الأحداث الرئيسة للشركات والاقتصاد.

<div id="available-groups">
  ## المجموعات المتاحة
</div>

| المجموعة | الوصف | نقطة النهاية |
| :--- | :--- | :--- |
| **Conference Calls** | جدول وتفاصيل مكالمات الشركات الجماعية (المؤتمرات الهاتفية) | [الرابط](/ar/api-reference/calendar_api/conference_calls/returns-conference-call-data) |
| **Dividends** | سجل وجدول مدفوعات التوزيعات | [الرابط](/ar/api-reference/calendar_api/dividends/dividends-events-v22) |
| **Earnings** | جدول تقارير أرباح الشركات | [الرابط](/ar/api-reference/calendar_api/earnings/returns-the-earnings-data) |
| **Economics** | تقويم الأحداث الاقتصادية العالمية | [الرابط](/ar/api-reference/calendar_api/economics/returns-the-economic-calendar-data) |
| **Guidance** | الإرشادات المالية للشركات | [الرابط](/ar/api-reference/calendar_api/guidance/returns-guidance-data) |
| **IPOs** | بيانات الطروحات العامة الأولية | [الرابط](/ar/api-reference/calendar_api/ipos/ipos-v21) |
| **Splits** | سجل وجدول تجزئة الأسهم | [الرابط](/ar/api-reference/calendar_api/splits/returns-the-splits-calendar-data) |
| **Ratings** | تصنيفات تفصيلية من المحللين | [الرابط](/ar/api-reference/calendar_api/ratings/returns-analyst-ratings-data) |
| **M&amp;A** | نشاط الاندماجات والاستحواذات | [الرابط](/ar/api-reference/calendar_api/ma/returns-the-mergers-and-acquisitions-data) |
