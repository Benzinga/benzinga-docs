<div id="introduction">
  ## المقدمة
</div>

توفر مجموعة واجهات برمجة التطبيقات Calendars إمكانية الوصول إلى مجموعة واسعة من بيانات التقويم المالي، مما يساعد المستثمرين على تتبع الأحداث الرئيسة للشركات والاقتصاد.

<div id="available-groups">
  ## المجموعات المتاحة
</div>

| المجموعة | الوصف | نقطة النهاية |
| :--- | :--- | :--- |
| **Conference Calls** | جدول وتفاصيل مكالمات الشركات الجماعية (المؤتمرات الهاتفية) | [الرابط](/ar/api-reference/calendar-api/get-conference-calls) |
| **Dividends** | سجل وجدول مدفوعات التوزيعات | [الرابط](/ar/api-reference/calendar-api/get-dividends) |
| **Earnings** | جدول تقارير أرباح الشركات | [الرابط](/ar/api-reference/calendar-api/get-earnings) |
| **Economics** | تقويم الأحداث الاقتصادية العالمية | [الرابط](/ar/api-reference/calendar-api/get-economics) |
| **Guidance** | الإرشادات المالية للشركات | [الرابط](/ar/api-reference/calendar-api/get-guidance) |
| **IPOs** | بيانات الطروحات العامة الأولية | [الرابط](/ar/api-reference/calendar-api/get-ipos) |
| **Splits** | سجل وجدول تجزئة الأسهم | [الرابط](/ar/api-reference/calendar-api/get-splits) |
| **Ratings** | تصنيفات تفصيلية من المحللين | [الرابط](/ar/api-reference/calendar-api/get-ratings) |
| **M&amp;A** | نشاط الاندماجات والاستحواذات | [الرابط](/ar/api-reference/calendar-api/get-ma) |
