<div id="introduction">
  ## المقدمة
</div>

تتعقّب واجهة برمجة التطبيقات Government Trades نشاط التداول للمسؤولين في الحكومة الأمريكية، بما في ذلك أعضاء مجلس النواب ومجلس الشيوخ الأمريكي. وهي توفر شفافية بشأن:

* تقارير المعاملات الدورية
* سجلات الصفقات الفردية
* أنواع الأصول ومبالغ الصفقات

<div id="available-endpoints">
  ## نقاط النهاية المتاحة
</div>

يُرجِع تداولات المسؤولين الحكوميين، بما في ذلك المعاملات التي يجريها أعضاء الكونغرس.
