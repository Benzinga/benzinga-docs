<div id="introduction">
  ## المقدّمة
</div>

تقوم واجهة برمجة التطبيقات Government Trades API بتتبّع نشاطات التداول التي يجريها المسؤولون في الحكومة الأمريكية، بما في ذلك أعضاء مجلس النواب ومجلس الشيوخ الأمريكي. وهي توفّر شفافية فيما يتعلّق بـ:

* التقارير الدورية للمعاملات
* سجلات التداول الفردية
* أنواع الأصول ومبالغ المعاملات

<div id="available-endpoints">
  ## نقاط النهاية المتاحة
</div>

<div id="get-government-trades">
  ### [جلب التداولات الحكومية](/ar/api-reference/calendar_api/government-trades/government-trades)
</div>

تُرجِع تداولات المسؤولين الحكوميين، بما في ذلك المعاملات التي يُجريها أعضاء الكونغرس.

<div id="get-government-trade-reports">
  ### [جلب تقارير التداولات الحكومية](/ar/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

تُرجع تقارير إفصاح مفصلة حول التداولات الحكومية.
