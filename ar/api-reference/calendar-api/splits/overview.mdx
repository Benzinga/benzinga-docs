<div id="introduction">
  ## المقدمة
</div>

واجهة برمجة التطبيقات API للتجزئات توفر معلومات حول عمليات تجزئة الأسهم التاريخية والمقبلة. وتتضمن تفاصيل مثل:

* نسبة التجزئة
* تاريخ الإعلان
* تاريخ الاستحقاق (Ex-date)
* تاريخ الدفع
