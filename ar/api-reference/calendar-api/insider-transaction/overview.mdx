<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات API لمعاملات المطلعين بيانات عن تداولات الأسهم التي يجريها المطلعون في الشركات، مثل المسؤولين التنفيذيين، وأعضاء مجلس الإدارة، وكبار المساهمين. وتتضمن بيانات مستمدة من ملفات نموذج SEC Form 4 المودعة.
