<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات Insider Transactions بيانات عن تداول الأسهم من قِبل المطلعين في الشركة، مثل المسؤولين والمديرين والمساهمين الرئيسيين. تتضمن هذه الواجهة بيانات مشتقة من نماذج Form 4 المقدَّمة إلى SEC.
