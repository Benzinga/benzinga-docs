<div id="introduction">
  ## المقدمة
</div>

واجهة برمجة التطبيقات API لـ NewsQuantified تُرجِع بيانات تحليلات إخبارية مُكمَّاة تضم درجات المعنويات، ومقاييس الصلة، ومؤشرات تأثير السوق. وهي توفّر بيانات مُنظَّمة تحلّل الجوانب النوعية للتغطية الإخبارية، بما في ذلك:

* قطبية المعنويات
* عدد المقالات
* درجات التوجّه (الترند)
* مقاييس خاصة بكل رمز تداول

تم تصميم هذه البيانات لتُستخدم في التداول الخوارزمي والبحوث الكمية.
