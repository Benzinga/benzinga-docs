<div id="introduction">
  ## المقدمة
</div>

توفّر واجهة برمجة التطبيقات API الخاصة بمحاضر مكالمات الأرباح إمكانية الوصول إلى النص الكامل للمحاضر والملفات الصوتية لمكالمات أرباح الشركات. وتتضمّن:

* النص الكامل لمحاضر مكالمات الأرباح ربع السنوية والسنوية
* جلسات الأسئلة والأجوبة مع تحديد هوية المتحدثين
* طوابع زمنية للتنقّل الدقيق
* عناوين url لتنزيل الملفات الصوتية
