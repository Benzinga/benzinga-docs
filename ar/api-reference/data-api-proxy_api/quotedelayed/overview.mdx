<div id="introduction">
  ## المقدمة
</div>

توفر واجهة برمجة التطبيقات API الخاصة بالأسعار المتأخرة عروض أسعار السوق بتأخير قدره 15 دقيقة لقائمة من الأوراق المالية. وهي تدعم الاستعلام عبر:

* الرموز
* أرقام التعريف الدولية للأوراق المالية ISIN
* CIK
