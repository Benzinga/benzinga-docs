<div id="introduction">
  ## المقدمة
</div>

تقوم واجهة برمجة التطبيقات API الخاصة بـ Bars بجلب بيانات الشموع السعرية التاريخية OHLCV (الافتتاح، الأعلى، الأدنى، الإغلاق، الحجم) للأوراق المالية المحددة. تُرجع بيانات أسعار تجميعية استنادًا إلى الفواصل الزمنية المحددة، وتدعم:

* رموز تداول متعددة
* نطاقات زمنية متنوعة (تواريخ مطلقة ونسبية)
* فترات زمنية مخصصة (1m، 5m، 1h، 1d، إلخ)
* تصفية جلسات التداول (جلسة ما قبل السوق، الجلسة العادية، جلسة ما بعد السوق)
