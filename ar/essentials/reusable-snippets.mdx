<div id="creating-a-custom-snippet">
  ## إنشاء مقتطف مخصص
</div>

**شرط مسبق**: يجب أن تنشئ ملف المقتطف الخاص بك داخل دليل `snippets`.

<Note>
  سيتم التعامل مع أي ملف في دليل `snippets` كمقتطف ولن يتم
  عرضه كصفحة مستقلة. إذا كنت تريد إنشاء صفحة مستقلة
  من المقتطف، فقم باستيراد المقتطف في ملف آخر واستدعِه كمكوّن.
</Note>

<div id="default-export">
  ### التصدير الافتراضي
</div>

1. أضف محتوى إلى ملف المقتطف الذي تريد إعادة استخدامه في عدة
   مواضع. ويمكنك اختياريًا إضافة متغيرات يمكن ملؤها عبر props
   عند استيراد المقتطف.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  يجب أن يكون المحتوى الذي تريد إعادة استخدامه داخل مجلد `snippets`
  لكي تعمل عملية الاستيراد.
</Warning>

2. استورد المقتطف إلى ملف الوجهة.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```


<div id="reusable-variables">
  ### متغيرات قابلة لإعادة الاستخدام
</div>

1. صدِّر متغيرًا من ملف المقتطف الخاص بك:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. قم باستيراد المقتطف من ملف الهدف لديك ثم استخدم المتغير:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```


<div id="reusable-components">
  ### المكوّنات القابلة لإعادة الاستخدام
</div>

1. داخل ملف الـ snippet الخاص بك، أنشئ مكوّناً يستقبل props عبر تصدير
   المكوّن كدالة سهمية (arrow function).

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  لا يُترجَم MDX داخل جسم الدالة السهمية (arrow function). التزم بصياغة HTML
  متى أمكن، أو استخدم `default export` إذا كنتَ بحاجة إلى استخدام MDX.
</Warning>

2. استورد القصاصة في ملف الهدف ومرِّر إليها الـ props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
