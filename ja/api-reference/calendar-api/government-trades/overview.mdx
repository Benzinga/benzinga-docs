<div id="introduction">
  ## はじめに
</div>

Government Trades API は、米国下院および上院議員を含む米国政府関係者による取引活動を追跡します。これにより、以下の点についての透明性が提供されます:

* 定期的な取引報告書
* 個別の取引記録
* 資産の種類および取引金額

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-government-trades">
  ### [政府取引を取得](/ja/api-reference/calendar_api/government-trades/government-trades)
</div>

連邦議会議員を含む政府高官による取引を返します。

<div id="get-government-trade-reports">
  ### [政府取引レポートの取得](/ja/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

詳細な政府取引開示レポートを返却します。
