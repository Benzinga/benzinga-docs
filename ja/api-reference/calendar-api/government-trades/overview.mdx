<div id="introduction">
  ## 概要
</div>

Government Trades API は、米国議会（下院・上院）議員を含む米国政府高官による取引活動を追跡します。これにより、以下の内容についての透明性が高まります：

* 定期的な取引報告書
* 個別の取引記録
* 資産の種類および取引金額

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-government-trades">
  ### [政府高官取引を取得](/ja/api-reference/calendar_api/government-trades/government-trades)
</div>

連邦議会議員による取引を含む、政府高官の取引を返します。

<div id="get-government-trade-reports">
  ### [政府取引レポートを取得](/ja/api-reference/calendar_api/government-trade-reports/government-trade-reports)
</div>

政府取引の詳細な開示レポートを返します。
