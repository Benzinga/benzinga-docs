<div id="introduction">
  ## 概要
</div>

Calendars API スイートは、幅広い金融カレンダー関連データへのアクセスを提供し、投資家が主要な企業および経済イベントを追跡できるようにします。

<div id="available-groups">
  ## 利用可能なグループ
</div>

| Group | Description | Endpoint |
| :--- | :--- | :--- |
| **Conference Calls** | 企業のカンファレンスコールのスケジュールと詳細 | [Link](/ja/api-reference/calendar-api/get-conference-calls) |
| **Dividends** | 配当金支払いの履歴とスケジュール | [Link](/ja/api-reference/calendar-api/get-dividends) |
| **Earnings** | 企業決算発表のスケジュール | [Link](/ja/api-reference/calendar-api/get-earnings) |
| **Economics** | 世界の経済イベントカレンダー | [Link](/ja/api-reference/calendar-api/get-economics) |
| **Guidance** | 企業の財務ガイダンス | [Link](/ja/api-reference/calendar-api/get-guidance) |
| **IPOs** | 新規株式公開（IPO）に関するデータ | [Link](/ja/api-reference/calendar-api/get-ipos) |
| **Splits** | 株式分割の履歴とスケジュール | [Link](/ja/api-reference/calendar-api/get-splits) |
| **Ratings** | 詳細なアナリスト評価 | [Link](/ja/api-reference/calendar-api/get-ratings) |
| **M&amp;A** | M&amp;A（合併・買収）活動 | [Link](/ja/api-reference/calendar-api/get-ma) |
