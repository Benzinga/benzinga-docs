<div id="introduction">
  ## 概要
</div>

Calendars API スイートは、幅広い金融カレンダー関連データへのアクセスを提供し、投資家が主要な企業および経済イベントを追跡できるようにします。

<div id="available-groups">
  ## 利用可能なグループ
</div>

| グループ                 | 説明                  | エンドポイント                                                 |
| :------------------- | :------------------ | :------------------------------------------------------ |
| **Conference Calls** | 企業のカンファレンスコールの予定と詳細 | [リンク](/ja/api-reference/calendar-api/get-conference-calls) |
| **Dividends**        | 配当金支払いの履歴と予定        | [リンク](/ja/api-reference/calendar-api/get-dividends)        |
| **Earnings**         | 企業の決算発表スケジュール       | [リンク](/ja/api-reference/calendar-api/get-earnings)         |
| **Economics**        | 世界の経済イベントカレンダー      | [リンク](/ja/api-reference/calendar-api/get-economics)        |
| **Guidance**         | 企業の業績見通し            | [リンク](/ja/api-reference/calendar-api/get-guidance)         |
| **IPOs**             | 新規株式公開データ           | [リンク](/ja/api-reference/calendar-api/get-ipos)             |
| **Splits**           | 株式分割の履歴と予定          | [リンク](/ja/api-reference/calendar-api/get-splits)           |
| **Ratings**          | 詳細なアナリスト評価情報        | [リンク](/ja/api-reference/calendar-api/get-ratings)          |
| **M&amp;A**          | 合併・買収の動向            | [リンク](/ja/api-reference/calendar-api/get-ma)               |
