<div id="introduction">
  ## はじめに
</div>

Insider Transactions API は、役員、取締役、大口株主などの企業インサイダーによる株式取引データを提供します。SEC の Form 4 提出書類に基づくデータも含まれます。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-insider-transaction-filing">
  ### [インサイダー取引届出書の取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

株式の買付、売却、オプション行使を含むインサイダー取引に関する届出書を返します。

<div id="get-insider-owner">
  ### [インサイダー保有者情報の取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

インサイダー保有者に関する情報を返します。
