<div id="introduction">
  ## はじめに
</div>

Insider Transactions API は、役員、取締役、大口株主などの企業インサイダーによる株式取引に関するデータを提供します。この API には、SEC の Form 4 提出書類から生成されたデータが含まれます。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-insider-transaction-filing">
  ### [インサイダー取引提出書類の取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-filing)
</div>

株式の買付、売却、オプション行使を含むインサイダー取引の提出書類を返します。

<div id="get-insider-owner">
  ### [Insider Owner の取得](/ja/api-reference/calendar_api/insider-transaction/insider-transaction-owner)
</div>

インサイダーオーナーの情報を返します。
