<div id="introduction">
  ## はじめに
</div>

Bulls Say Bears Say API は、指定した株式の ticker シンボルに対する最新の強気・弱気の投資ケースを返します。

* **強気ケース (Bull Cases)**: 株式を購入すべきとする肯定的な主張を提示します。
* **弱気ケース (Bear Cases)**: その株式への投資に反対する否定的な主張を提示します。

このデータは、有価証券の将来パフォーマンスに関する相反する見解を理解するうえで有用です。
