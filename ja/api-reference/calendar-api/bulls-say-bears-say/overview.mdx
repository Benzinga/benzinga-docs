<div id="introduction">
  ## はじめに
</div>

Bulls Say Bears Say API は、指定した株式 ticker シンボルに対する最新の強気・弱気の投資ケースを返します。

* **Bull Cases**: 株式を購入することを支持する肯定的な根拠を示します。
* **Bear Cases**: その株式への投資に反対する否定的な根拠を示します。

このデータは、ある証券の将来のパフォーマンスに関する対立する見解を理解するうえで有用です。
