<div id="introduction">
  ## 概要
</div>

Splits API は、過去および今後予定されている株式分割に関する情報を提供します。以下のような詳細が含まれます：

* 分割比率
* 発表日
* 権利落ち日
* 効力発生日

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-splits">
  ### [株式分割を取得](/ja/api-reference/calendar-api/get-splits)
</div>

株式分割のカレンダーデータを返します。
