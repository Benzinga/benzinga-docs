<div id="introduction">
  ## 概要
</div>

Splits API は、過去および今後予定されている株式分割に関する情報を提供します。以下のような詳細が含まれます：

* 分割比率
* 発表日
* 権利落ち日
* 効力発生日

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-splits">
  ### [Splits 取得](/ja/api-reference/calendar_api/splits/returns-the-splits-calendar-data)
</div>

Splits のカレンダーデータを返します。
