<div id="introduction">
  ## 概要
</div>

Splits API は、過去および今後予定されている株式分割に関する情報を提供します。主な項目は次のとおりです。

* 分割比率
* 発表日
* 権利落ち日
* 支払日

  ## 利用可能なエンドポイント
</div>

  ### [Splitsの取得](/ja/api-reference/calendar_api/splits/returns-the-splits-calendar-data)
</div>

株式分割カレンダーのデータを返します。
