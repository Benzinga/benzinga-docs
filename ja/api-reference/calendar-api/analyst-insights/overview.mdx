<div id="introduction">
  ## はじめに
</div>

Analyst Insights API は、金融アナリストによる詳細な分析や推奨を含む、有価証券に関するリサーチ見解を返します。これにより、以下の情報にアクセスできます。

* アナリストによる格上げ・格下げ
* 目標株価
* 詳細なリサーチノート
* コンセンサスレーティングの背景情報

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-analyst-insights">
  ### [アナリスト・インサイトの取得](/ja/api-reference/calendar_api/analyst-insights/analyst-insights-v1)
</div>

特定の証券、企業、またはアナリストに関するアナリスト・インサイトおよびリサーチ見解を取得します。
