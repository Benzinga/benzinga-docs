<div id="introduction">
  ## はじめに
</div>

マーケットデータAPI は、証券に関する Delayed Quotes、マーケットムーバー、空売り残高データなど、幅広い市場情報を提供します。

<div id="market-movers">
  ## 値動き上位銘柄
</div>

マーケットムーバー API は、値上がり率上位、値下がり率上位、および出来高上位の証券に関するリアルタイムデータを提供します。このエンドポイントを使用すると、次を特定できます。

* **値上がり銘柄** - 値上がり幅が最も大きい証券
* **値下がり銘柄** - 値下がり幅が最も大きい証券
* **出来高上位銘柄** - 売買高が最も多い証券

各結果には、価格、変動率、出来高、時価総額、セクター、浮動株数のデータが含まれます。

<div id="get-market-movers">
  ### [マーケットムーバーを取得](/ja/api-reference/market-data/get-market-movers)
</div>

値上がり銘柄、値下がり銘柄、出来高上位銘柄などの主要なマーケットムーバーを取得します。

<div id="short-interest-data">
  ## 空売り残高データ
</div>

Short Interest APIは、証券の空売りデータを提供します。このデータは、市場センチメントの把握や、ショートスクイーズ候補の特定に役立ちます。

主な指標は次のとおりです。

* **総空売り残高** - 空売りされた株式数
* **カバー日数** - 平均出来高に基づく、空売りポジションの買い戻しに必要と見積もられる日数
* **浮動株に占める空売り比率** - 浮動株のうち空売りされている割合
* **前月比の変化** - 前月からの空売り残高の変化

  ### [空売り残高データを取得](/ja/api-reference/market-data/get-short-interest-data)
</div>

証券の空売り残高データを取得します。
