<div id="introduction">
  ## はじめに
</div>

Earning Ratios API は、企業の利益や収益性に関連する比率を提供します。たとえば、次のようなものがあります。

* ROA (総資産利益率)
* ROE (自己資本利益率)
* ROI (投資利益率)

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-earning-ratios">
  ### [Earning Ratiosを取得](/ja/api-reference/earningratios/get-earning-ratios)
</div>

指定したシンボル一覧のEarning Ratiosを取得します。
