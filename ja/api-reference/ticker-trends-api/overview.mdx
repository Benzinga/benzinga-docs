<div id="introduction">
  ## 概要
</div>

Ticker Trends API は、ユーザーのエンゲージメント指標に基づいて、指定した ticker シンボルのトレンドデータを取得します。これにより、次の項目を追跡することで関心の高まりを示す有価証券を特定できます：

* トレンドスコア
* アクティビティレベル
* さまざまな時間間隔でのエンゲージメント

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-ticker-trend-data">
  ### [ティッカー・トレンドデータの取得](/ja/api-reference/ticker-trends-api_api/ticker-trends/get-ticker-trend-data)
</div>

指定したティッカーについて集計したトレンドスコアを取得します。

<div id="get-trending-tickers-list">
  ### [トレンドティッカー一覧を取得](/ja/api-reference/ticker-trends-api_api/ticker-trends/get-ticker-trend-list-data)
</div>

トレンドのティッカーをランキング順の一覧として取得します。
