<div id="introduction">
  ## はじめに
</div>

Bars API は、指定した証券の過去の OHLCV (始値、高値、安値、終値、出来高) 価格バー・データを取得します。指定した間隔で集計された価格データを返し、以下をサポートします。

* 複数のティッカーシンボル
* さまざまな時間範囲 (絶対日付と相対日付)
* カスタマイズ可能な時間バケット (1m、5m、1h、1d など)
* 取引セッションのフィルタリング (プレマーケット、通常取引時間、時間外取引)
