<div id="introduction">
  ## はじめに
</div>

この REST API では、カンファレンスコールとそのサマリーを取得するためのエンドポイントを提供しています。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="fetch-all-calls">
  ### [すべてのコールの取得](/ja/api-reference/delivery_api/calls/fetch-all-calls)
</div>

オプションのフィルタリングとページネーションを用いて、コールの一覧を取得します。

<div id="fetch-call-by-id">
  ### [Call を ID で取得](/ja/api-reference/delivery_api/calls/fetch-call-by-id)
</div>

特定の Call の ID を指定して詳細情報を取得します。

<div id="fetch-summaries">
  ### [要約の取得](/ja/api-reference/delivery_api/summaries/fetch-summaries)
</div>

ページネーション対応の要約一覧を取得します。

<div id="fetch-summary-by-call-id">
  ### [Fetch Summary by Call ID](/ja/api-reference/delivery_api/summaries/fetch-summary-by-call-id)
</div>

特定のコールIDに対応するサマリー情報を取得します。
