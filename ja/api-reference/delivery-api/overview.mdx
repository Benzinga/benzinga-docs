<div id="introduction">
  ## はじめに
</div>

この REST API では、カンファレンスコールおよび要約を取得するためのエンドポイントを提供しています。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="fetch-all-calls">
  ### [すべてのコールを取得](/ja/api-reference/delivery_api/calls/fetch-all-calls)
</div>

任意のフィルタリングおよびページネーションを指定して、コールの一覧を取得します。

<div id="fetch-call-by-id">
  ### [ID でコールを取得](/ja/api-reference/delivery_api/calls/fetch-call-by-id)
</div>

特定のコールについて、ID を指定して詳細情報を取得します。

<div id="fetch-summaries">
  ### [要約一覧の取得](/ja/api-reference/delivery_api/summaries/fetch-summaries)
</div>

ページネーション対応の要約一覧を取得します。

<div id="fetch-summary-by-call-id">
  ### [Call ID でサマリーを取得](/ja/api-reference/delivery_api/summaries/fetch-summary-by-call-id)
</div>

特定の Call ID に対応するサマリーの詳細を取得します。
