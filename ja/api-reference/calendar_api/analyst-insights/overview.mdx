<div id="introduction">
  ## はじめに
</div>

Analyst Insights API は、証券に関するアナリストの見解を返し、金融アナリストによる詳細な分析および推奨情報を提供します。これにより、以下の情報にアクセスできます。

* アナリストによる格上げおよび格下げ
* 目標株価
* 詳細なリサーチノート
* コンセンサスレーティングに関する背景情報

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-analyst-insights">
  ### [アナリスト・インサイトを取得](/ja/api-reference/calendar_api/analyst-insights/get-analyst-insights-v1)
</div>

特定の証券、金融機関、またはアナリストに関するアナリストのインサイトおよびリサーチ見解を取得します。
