<div id="introduction">
  ## はじめに
</div>

企業ファンダメンタルズAPIでは、企業を詳細に分析するために必要な包括的な財務データを取得できます。対象は以下のとおりです。

* **財務諸表**: 貸借対照表、損益計算書、キャッシュフロー計算書 (オリジナル、修正版、TTM) 。
* **企業プロフィール**: 事業内容、セクター、業種、経営陣、本社所在地。
* **決算情報**: 過去の売上高、EPS、EBITDA、純利益。
* **株式クラスとバリュエーション**: 株式クラスの詳細、アルファ/ベータ、バリュエーション指標。
