<div id="introduction">
  ## はじめに
</div>

Operation Ratios API は、企業の業務効率を評価するための各種指標を提供します。たとえば、次のようなものがあります。

* 総資産回転率
* 棚卸資産回転率
* 売掛金回転率

  ## 利用可能なエンドポイント
</div>

<div id="get-operation-ratios">
  ### [Operation Ratiosを取得](/ja/api-reference/operationratios/get-operation-ratios)
</div>

指定したシンボル一覧のOperation Ratiosを取得します。
