<div id="introduction">
  ## はじめに
</div>

Quotes Delayed API（Delayed Quotes と同様）は、15 分遅延したマーケットクオートを提供します。

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-delayed-quotes">
  ### [遅延気配の取得](/ja/api-reference/data-api-proxy_api/quotedelayed/get-delayed-quotes-for-a-list-of-symbols)
</div>

シンボルのリストに対する遅延気配を取得します。
