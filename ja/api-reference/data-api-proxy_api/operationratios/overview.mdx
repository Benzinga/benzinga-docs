<div id="introduction">
  ## はじめに
</div>

Operation Ratios API は、企業の業務効率を評価するための各種指標を提供します。主な指標には次のようなものがあります。

* 総資産回転率
* 棚卸資産回転率
* 売掛金回転率

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-operation-ratios">
  ### [Operation Ratios を取得](/ja/api-reference/data-api-proxy_api/operationratios/get-operation-ratios-v21)
</div>

複数のシンボルに対する Operation Ratios を取得します。
