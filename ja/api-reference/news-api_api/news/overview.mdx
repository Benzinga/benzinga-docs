<div id="introduction">
  ## はじめに
</div>

Benzinga Newsfeed API は、「Why Is It Moving」（WIIM）機能を含む、さまざまなコンテンツタイプのリアルタイム金融ニュースを提供します。柔軟なフィルタリングが可能で、以下の条件で絞り込みをサポートします:

* ティッカー、ISIN（国際証券識別番号）、CUSIP
* チャンネルとトピック
* 日付範囲と更新日時
* コンテンツタイプと重要度

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-news">
  ### [ニュースを取得](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

ニュース記事の構造化されたデータを返します。

<div id="get-news-channels">
  ### [ニュースチャネルの取得](/ja/api-reference/news-api_api/channels/get-available-news-channels)
</div>

フィルタリングに利用可能なすべてのニュースチャネルの一覧を返します。

<div id="get-removed-news">
  ### [削除済みニュースを取得](/ja/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

削除または訂正されたニュース記事を返します。

<div id="why-is-it-moving-wiim">
  ### [Why Is It Moving (WIIM)](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

対象となる株価の値動きについて、Benzinga 独自の一文解説にアクセスできます。これらの項目を取得するには、標準のニュースエンドポイントを使用してください。
