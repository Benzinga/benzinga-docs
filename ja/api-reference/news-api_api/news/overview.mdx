<div id="introduction">
  ## はじめに
</div>

Benzinga Newsfeed API は、以下の条件で柔軟にフィルタリングできるリアルタイム金融ニュースを提供します:

* ティッカー、ISIN（国際証券識別番号）、CUSIP
* チャンネルとトピック
* 日付範囲と更新日時
* コンテンツタイプと重要度

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-news">
  ### [ニュースを取得](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

ニュース記事の構造化されたデータを返します。

<div id="get-news-channels">
  ### [ニュースチャネルの取得](/ja/api-reference/news-api_api/channels/get-available-news-channels)
</div>

フィルタリングに利用可能なすべてのニュースチャネルの一覧を返します。

<div id="get-removed-news">
  ### [削除済みニュースを取得](/ja/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

削除または訂正されたニュース記事を返します。
