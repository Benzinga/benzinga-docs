<div id="introduction">
  ## はじめに
</div>

Bars API は、指定した有価証券について、過去の OHLCV（始値・高値・安値・終値・出来高）バー・データ（価格足）を取得します。指定したインターバルごとに集計した価格データを返し、次の機能をサポートします。

* 複数の ticker シンボル
* さまざまな期間（絶対日付および相対日付）
* カスタマイズ可能な時間バケット（1m, 5m, 1h, 1d など）
* 取引セッション（プレマーケット、レギュラー、アフターマーケット）によるフィルタリング
