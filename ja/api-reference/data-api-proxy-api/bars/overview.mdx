<div id="introduction">
  ## 概要
</div>

Bars API は、指定した証券の過去の OHLCV（始値、高値、安値、終値、出来高）価格バー データを取得します。指定したインターバルに基づいて集計された価格データを返し、次の機能をサポートします：

* 複数の ticker シンボル
* さまざまな期間（絶対日付および相対日付）
* カスタマイズ可能な時間バケット（1m、5m、1h、1d など）
* 取引セッションのフィルタリング（プレマーケット、レギュラー、アフターマーケット）
