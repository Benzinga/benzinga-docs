<div id="introduction">
  ## はじめに
</div>

Earning Ratios API は、企業の収益性や利益率に特化した指標を提供します。たとえば、次のような比率があります：

* ROA (Return on Assets / 総資産利益率)
* ROE (Return on Equity / 自己資本利益率)
* ROI (Return on Investment / 投資収益率)
