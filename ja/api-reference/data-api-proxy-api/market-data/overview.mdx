<div id="introduction">
  ## はじめに
</div>

Market Data API 群は、遅延株価、マーケットムーバー、証券の空売り残高データを含む網羅的な市場情報を提供します。

<div id="market-movers">
  ## マーケットムーバー
</div>

Market Movers API は、最も値上がりしている銘柄、値下がりしている銘柄、最も活発に取引されている銘柄に関するリアルタイムデータを提供します。このエンドポイントを使用すると、次の種類の銘柄を特定できます：

* **Gainers** - 価格の上昇率が最も大きい銘柄
* **Losers** - 価格の下落率が最も大きい銘柄
* **Volume Leaders** - 出来高が最も多い銘柄

各結果には、価格、変化率、出来高、時価総額、セクター、およびフロート（浮動株）に関するデータが含まれます。

<div id="get-market-movers">
  ### [マーケットムーバーの取得](/ja/api-reference/data-api-proxy_api/market-data/market-movers)
</div>

上昇銘柄、下落銘柄、出来高上位銘柄など、トップのマーケットムーバーを取得します。

<div id="short-interest-data">
  ## 空売り残高データ
</div>

Short Interest API は、有価証券の空売り取引に関するデータを提供します。このデータは、市場センチメントを把握し、ショートスクイーズ候補を特定するうえで有用です。

主な指標は以下のとおりです。

* **Total Short Interest** - 空売りされた株式数
* **Days to Cover** - 平均出来高に基づいて空売りポジションを解消するのに必要と推定される日数
* **Short % of Float** - フロート株（流通株）のうち空売りされている割合
* **Month-over-Month Change** - 前月からの空売り残高の増減
