<div id="introduction">
  ## はじめに
</div>

Valuation Ratios API は、証券の相対的な価値（割高・割安）を評価するための主要なバリュエーション指標を提供します。これには、次のような比率が含まれます：

* P/E（株価収益率：Price-to-Earnings）
* P/S（株価売上高倍率：Price-to-Sales）
* P/B（株価純資産倍率：Price-to-Book）
