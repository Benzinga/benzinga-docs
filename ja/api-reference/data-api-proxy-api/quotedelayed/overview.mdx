<div id="introduction">
  ## はじめに
</div>

Delayed Quotes API は、複数の証券について 15 分遅れの市場気配情報を提供します。以下の条件での取得をサポートします。

* シンボル
* ISIN（国際証券識別番号）
* CIK（SEC 企業識別子）
