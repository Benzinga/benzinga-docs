<div id="introduction">
  ## はじめに
</div>

Delayed Quotes API は、複数の証券について 15 分遅延の市場気配値を提供します。以下の指定方法での取得をサポートします:

* シンボル
* ISIN（国際証券識別番号）
* CIK（SEC の企業識別子）
