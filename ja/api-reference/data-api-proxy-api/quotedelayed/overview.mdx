<div id="introduction">
  ## はじめに
</div>

Delayed Quotes API は、証券のリストに対して 15 分遅延したマーケットクオートを提供します。以下の方法での取得をサポートします:

* シンボル
* ISIN (国際証券識別番号) 
* CIK (SEC 企業識別子)
