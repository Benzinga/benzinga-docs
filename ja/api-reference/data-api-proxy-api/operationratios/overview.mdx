<div id="introduction">
  ## 概要
</div>

Operation Ratios API は、以下のような企業のオペレーション効率を評価するための指標を提供します：

* 資産回転率
* 在庫回転率
* 売掛金回転率

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-operation-ratios">
  ### [営業比率の取得](/ja/api-reference/data-api-proxy_api/operationratios/get-operation-ratios-v21)
</div>

シンボルのリストに対する営業比率を取得します。
