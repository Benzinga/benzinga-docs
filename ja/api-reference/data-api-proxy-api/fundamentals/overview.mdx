<div id="introduction">
  ## 概要
</div>

Company Fundamentals API は、企業を深く分析するために必要な包括的な財務データを取得します。以下をカバーします。

* **財務諸表**: 貸借対照表、損益計算書、キャッシュ・フロー計算書（オリジナル、修正後、TTM）。
* **企業プロフィール**: 事業内容、セクター、業種、経営陣、本社所在地。
* **決算情報**: 過去の売上高、EPS、EBITDA、純利益。
* **株式クラスとバリュエーション**: 株式クラスの詳細、アルファ/ベータ、バリュエーション指標（各種比率）。
