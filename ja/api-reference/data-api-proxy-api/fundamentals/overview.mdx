<div id="introduction">
  ## はじめに
</div>

Company Fundamentals API は、企業を詳細に分析するために必要な包括的なファンダメンタルズ（財務）データを取得します。対象範囲は次のとおりです。

* **財務諸表**: 貸借対照表、損益計算書、キャッシュ・フロー計算書（オリジナル、再表示、TTM）。
* **企業プロフィール**: 事業内容、セクター、業種、経営陣、本社所在地。
* **決算情報**: 過去の売上高、EPS、EBITDA、純利益。
* **株式クラスとバリュエーション**: 株式クラスの詳細、アルファ/ベータ、およびバリュエーション指標。
