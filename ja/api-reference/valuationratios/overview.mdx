<div id="introduction">
  ## はじめに
</div>

Valuation Ratios API は、証券の相対的な価値を評価するための主要なバリュエーション指標を提供します。具体的には、次のような比率が含まれます。

* P/E (株価収益率)
* P/S (株価売上高倍率)
* P/B (株価純資産倍率)

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-valuation-ratios">
  ### [Valuation Ratiosを取得](/ja/api-reference/valuationratios/get-valuation-ratios)
</div>

指定したシンボル一覧のValuation Ratiosを取得します。
