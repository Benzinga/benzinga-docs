<div id="introduction">
  ## はじめに
</div>

NewsQuantified API は、センチメントスコア、関連性指標、市場インパクト指標などを含む定量化されたニュース分析データを返します。ニュース報道の質的な側面を分析した構造化データを提供し、次の内容を含みます:

* センチメントの極性
* 記事数
* トレンドスコア
* ticker ごとの指標

このデータは、アルゴリズム取引およびクオンツリサーチ向けに設計されています。
