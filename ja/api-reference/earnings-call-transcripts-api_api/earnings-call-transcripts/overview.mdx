<div id="introduction">
  ## はじめに
</div>

Earnings Call Transcripts API は、企業の決算説明会の全文テキストおよび音声ファイルへのアクセスを提供します。これには以下が含まれます：

* 四半期および通期の決算説明会の全文テキスト書き起こし
* Q&amp;A セッションおよび話者の識別情報
* 詳細なナビゲーションのためのタイムスタンプ
* ダウンロード可能な音声ファイルの URL

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-earnings-call-transcripts">
  ### [決算説明会トランスクリプトを取得](/ja/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcripts)
</div>

指定した企業またはコールIDに対応する決算説明会トランスクリプトを取得します。

<div id="get-earnings-call-audio">
  ### [決算説明会音声を取得](/ja/api-reference/earnings-call-transcripts-api_api/earnings-call-transcripts/get-earnings-call-transcript-audio-files)
</div>

決算説明会の音声全編の録音をダウンロードするための署名付きURLを取得します。
