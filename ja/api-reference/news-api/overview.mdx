<div id="introduction">
  ## はじめに
</div>

Benzinga Newsfeed API は、以下の条件に基づいて柔軟にフィルタリングできるリアルタイムの金融ニュースを提供します。

* ティッカー、ISIN（国際証券識別番号）、CUSIP
* チャンネルおよびトピック
* 日付範囲および更新日時
* コンテンツタイプおよび重要度

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-news">
  ### [ニュースを取得](/ja/api-reference/news-api_api/news/get-the-news-items)
</div>

ニュースアイテムの構造化データを返します。

<div id="get-news-channels">
  ### [ニュースチャンネル一覧を取得](/ja/api-reference/news-api_api/channels/get-available-news-channels)
</div>

フィルタリングに使用可能なすべてのニュースチャンネルの一覧を返します。

<div id="get-removed-news">
  ### [削除済みニュースを取得](/ja/api-reference/news-api_api/news/returns-the-removed-news-data)
</div>

削除または修正されたニュース記事を返します。
