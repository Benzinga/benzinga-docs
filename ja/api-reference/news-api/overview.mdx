<div id="introduction">
  ## はじめに
</div>

Benzinga Newsfeed API は、以下の条件で柔軟にフィルタリングできるリアルタイム金融ニュースを提供します:

* ティッカー、ISIN (国際証券識別番号) 、CUSIP
* チャンネルとトピック
* 日付範囲と更新日時
* コンテンツタイプと重要度

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

<div id="get-news">
  ### [ニュースを取得](/ja/api-reference/news-api/get-news-items)
</div>

ニュース記事の構造化されたデータを返します。

<div id="get-news-channels">
  ### [ニュースチャンネルの取得](/ja/api-reference/news-api/channels/get-available-news-channels)
</div>

フィルタリングに利用可能なすべてのニュースチャンネルの一覧を返します。

<div id="get-removed-news">
  ### [削除済みニュースを取得](/ja/api-reference/news-api/get-removed-news)
</div>

削除または訂正されたニュース記事を返します。
