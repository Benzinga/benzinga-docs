<div id="introduction">
  ## はじめに
</div>

Why Is It Moving（WIIMs）は、「なぜこの株は動いているのか？」という重要な問いに答えるための特化したニュースフィードです。厳選されたコンテンツにより、有意な株価変動の背景にあるきっかけや要因を説明するニュースと分析を文脈とともに提供し、市場の動きの「なぜ」をリアルタイムで理解できるようにします。

<div id="key-features">
  ## 主な特長
</div>

* **コンテクスチュアルニュース**: 株価の値動きの背景を分かりやすく解説することに特化したニュース記事
* **タイムリーなカバレッジ**: 直近の株価変動を引き起こしているイベントに関するリアルタイムニュース
* **投資判断に直結するインサイト**: サプライズ決算、アナリストの格上げ・格下げ、異常なオプション取引、速報的な進展など、市場を動かすカタリストの分析
* **厳選されたコンテンツ**: 最も重要な市場を動かすイベントを強調するようにフィルタリングされたニュース

<div id="use-cases">
  ## 利用例
</div>

* **マーケット動向の把握**: 予想外の価格変動の背景要因を素早く理解する
* **トレーディングインテリジェンス**: 取引判断を下す前に、材料となるニュースにアクセスする
* **ポートフォリオ監視**: 保有銘柄に影響を与えるニュースを継続的に把握しておく
* **マーケットリサーチ**: セクターごとに株価変動を引き起こす要因のパターンを追跡する
