<div id="introduction">
  ## はじめに
</div>

Why Is It Moving (WIIMs) は、「なぜこの銘柄は動いているのか？」という重要な問いに答えるための特化したニュースフィードです。この厳選されたコンテンツは、株価の大きな変動を引き起こすカタリストを説明する文脈付きのニュースと分析を提供し、市場の動きの「なぜ」をリアルタイムで理解できるようにします。

<div id="key-features">
  ## 主な特徴
</div>

* **文脈付きニュース**: 株価の値動きの背景をわかりやすく解説することに特化したストーリー
* **タイムリーなカバレッジ**: 直近の価格変動を引き起こしている要因に関するリアルタイムニュース
* **アクションにつながるインサイト**: 決算サプライズ、アナリストの格上げ・格下げ、異常なオプション取引、速報的な新展開など、市場を動かす材料の分析
* **キュレーションされたコンテンツ**: 市場インパクトの大きい重要イベントを厳選して伝えるニュース

<div id="use-cases">
  ## ユースケース
</div>

* **マーケットの動きの把握**: 想定外の価格変動が起きた理由を素早く理解する
* **トレーディング・インテリジェンス**: 取引判断の前に、価格変動のきっかけとなる要因を解説するニュースへアクセスする
* **ポートフォリオ・モニタリング**: 保有銘柄に影響を与えるニュースを継続的に把握する
* **マーケット・リサーチ**: セクター横断で株価の動きを左右する要因のパターンを追跡する
