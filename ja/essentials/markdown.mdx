<div id="titles">
  ## タイトル
</div>

セクション見出しとして使うのが最適です。

```md
<div id="titles">
  ## タイトル
</div>

```


<div id="subtitles">
  ### サブタイトル
</div>

セクション内の小見出しに最適です。

```md
<div id="subtitles">
  ### サブタイトル
</div>

```

<Tip>
  各 **title** と **subtitle** からアンカーが作成され、右側の目次にも表示されます。
</Tip>


<div id="text-formatting">
  ## テキストの書式設定
</div>

ほとんどの Markdown 記法による書式設定をサポートしています。テキストを `**`、`_`、`~` で囲むことで装飾できます。

| スタイル        | 書き方               | 結果                |
| ------------- | ----------------- | --------------- |
| 太字           | `**bold**`        | **bold**        |
| 斜体           | `_italic_`        | _italic_        |
| 取り消し線        | `~strikethrough~` | ~strikethrough~ |

これらを組み合わせることもできます。たとえば、`**_bold and italic_**` と書くと、**_bold and italic_** のようなテキストになります。

上付き文字と下付き文字を記述するには HTML を使用する必要があります。テキストの前後を `<sup>` または `<sub>` で囲んでください。

| 文字種別       | 書き方                        | 結果                      |
| ----------- | ------------------------ | ---------------------- |
| 上付き文字      | `<sup>superscript</sup>` | <sup>superscript</sup> |
| 下付き文字      | `<sub>subscript</sub>`   | <sub>subscript</sub>   |

<div id="linking-to-pages">
  ## ページへのリンク
</div>

リンクにしたいテキストを `[]()` で囲むことでリンクを追加できます。たとえば、`[link to google](https://google.com)` と書くと [link to google](https://google.com) になります。

ドキュメント内のページへのリンクはルート相対パスにする必要があります。基本的には、フォルダーのパス全体を含めてください。たとえば、`[link to text](/writing-content/text)` はコンポーネントセクション内の "Text" ページへのリンクになります。

`[link to text](../text)` のような相対リンクは、最適化がしづらいため表示が遅くなります。

  ## 引用ブロック
</div>

<div id="singleline">
  ### 1行
</div>

ブロック引用を作成するには、段落の先頭に `>` を付けます。

> ドロシーは、彼女の城にある多くの美しい部屋を、彼女のあとについて歩いて回った。

```md
> ドロシーは彼女の後について、城の美しい部屋をいくつも通り抜けた。
```


<div id="multiline">
  ### 複数行
</div>

> ドロシーは、魔女のあとについて、彼女の城のいくつもの美しい部屋を通って歩いていった。
>
> 魔女は、鍋ややかんをきれいにし、床を掃き、薪をくべて火を絶やさないようにするよう彼女に命じた。

```md
> ドロシーは彼女の後について城の美しい部屋を数多く通り抜けた。
>
> 魔女は彼女に鍋や釜を洗い、床を掃き、薪で火を絶やさないよう命じた。
```
