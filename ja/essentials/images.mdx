<div id="image">
  ## 画像
</div>

<div id="using-markdown">
  ### Markdown を使用する
</div>

[Markdown 構文](https://www.markdownguide.org/basic-syntax/#images)を使うと、以下のコードで画像を追加できます。

```md
![title](/path/image.jpg)
```

画像ファイルのサイズは 5MB 未満である必要があります。5MB を超える場合は、[Cloudinary](https://cloudinary.com/) や [S3](https://aws.amazon.com/s3/) などのサービスでホスティングすることを推奨します。その URL を指定して埋め込んでください。

<div id="using-embeds">
  ### 埋め込みの使用
</div>

画像をより柔軟にカスタマイズしたい場合は、[embeds](/ja/writing-content/embed) を使用して画像を追加することもできます。

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

ドキュメント内に他の HTML ページを読み込みます。主に動画を埋め込むために使用されます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
