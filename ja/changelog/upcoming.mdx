---
title: "今後の予定"
---

今後のリリース内容と、次回以降のアップデートに向けて予定しているアクション項目です。

<div id="december-2024">
  ## 2024年12月
</div>

- **[Dynamic Event Feedback API](https://calendarapi.com/docs/event-feedback) のリリースを予定**しており、ユーザーがカレンダーインターフェースから直接イベントを評価およびレビューできるようにします。
- **一括イベント作成のパフォーマンスを強化**し、API のレスポンス時間を 20% 削減することを目標としています。
- **複数リージョンのフェイルオーバー対応を API ダッシュボードに導入**し、サービスの信頼性を向上させます。
- **リアルタイム出席状況インサイト機能を開発**しており、参加者の動向をライブで監視できるウェブソケット更新を提供します。
- **[Advanced Webhook Security](https://calendarapi.com/docs/webhooks/security) の提供を開始**し、シグネチャ検証および IP ホワイトリスト登録をサポートします。
- **エンタープライズ顧客向けに CalendarAPI Premium Managed Services を順次展開**し、24時間365日のサポートとカスタム SLA を提供します。
- **CDN 連携を最適化**し、ICS やイベント画像といったアセットのロード時間を改善します。

---

<div id="january-2025">
  ## 2025年1月
</div>

- **AI を活用したイベントレコメンデーション機能のサポートを追加**し、ユーザーの好みとカレンダー上のアクティビティに合わせて最適化します。
- **Calendar 同期デバッガーツールを導入**し、開発者が同期の問題をより効率的に特定・解決できるようにします。
- **API キー管理ツールを拡充**し、きめ細かなアクセス制御や自動有効期限ポリシーを追加します。
- **[Event Cloning API](https://calendarapi.com/docs/event-cloning) を提供開始**し、複雑なイベント設定の迅速な複製を可能にします。
- **CalendarAPI サンドボックス環境を公開**し、開発者がライブデータに影響を与えることなく API をテストできるようにします。
- **ドキュメントを強化**し、追加のユースケースガイドやインタラクティブな API ウォークスルーを提供します。
