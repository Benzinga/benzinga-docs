<AccordionGroup>
    <Accordion title="Benzinga とは何ですか？また、どのような種類の API を提供していますか？">
        Benzinga は、API を通じて多様な金融データおよびニュースサービスを提供するフィンテック企業です。これらの API は、リアルタイムおよび過去の市場データ、ニュース、企業のファンダメンタルズ、トレーディングシグナルなど、幅広い情報をカバーしており、企業が自社アプリケーションに金融データを直接組み込めるようにします。
    </Accordion>

    <Accordion title="Benzinga の API の主な特長は何ですか？">
        Benzinga の API は、以下を提供します:
        - **リアルタイム市場データ**: 気配値、約定などを含むライブデータストリームへのアクセス。
        - **過去データ**: 過去の価格や出来高を取得可能。
        - **ニュースとインサイト**: 複数のソースから集約されたニュース、記事、金融インサイト。
        - **企業ファンダメンタルズ**: 企業の財務情報、アナリストレーティング、主要指標に関する情報。
        - **トレーディングシグナルとアラート**: 市場状況に基づくカスタマイズ可能なアラート。
    </Accordion>

    <Accordion title="ニュースフィードはどのくらいの頻度で更新されますか？">
        Benzinga のニュースフィード API は、株価の動きや市場トレンドに関するタイムリーなニュースを利用者が受け取れるよう、数分おきに更新されます。この API には、トップストーリー、株価変動の要因、市場アップデートを表示するためのオプションが含まれます。
    </Accordion>

    <Accordion title="Benzinga はリアルタイムデータ向けに WebSocket をサポートしていますか？">
        はい。Benzinga はリアルタイムのデータ配信向けに WebSocket（ウェブソケット）API を提供しています。この API は、気配値、約定、その他のデータポイントを含む継続的なライブ市場データフィードを必要とするアプリケーションに最適です。特定のデータ要件に応じて WebSocket 接続を構成できます。
    </Accordion>

    <Accordion title="どのような種類の過去データが利用できますか？">
        Benzinga の過去データ API では、日次、週次、月次、および日中足のバー・データなど、株式、指数、その他のアセットを対象としたさまざまなデータにアクセスできます。アセットの種類や必要な詳細レベルに応じて、数年分さかのぼって過去データを取得することが可能です。
    </Accordion>

    <Accordion title="Benzinga の API を通じて決算レポートやその他の企業イベントにアクセスできますか？">
        はい。Benzinga は、今後および過去の決算発表、IPO、配当発表、その他の企業イベントを含む Calendar API を提供しています。これにより、投資に影響する重要な金融イベントを常に把握できます。
    </Accordion>

    <Accordion title="API 利用に制限はありますか？">
        各 API サブスクリプションには、一定時間内に実行できるリクエスト数を制限するレートリミットが設定されています。これらの制限は、対象の API とサブスクリプションのティアによって異なります。ご利用のプランに関連する制限については、Benzinga のサポートチームに問い合わせるか、ドキュメントを参照してください。
    </Accordion>

    <Accordion title="トライアル版やサンドボックス環境は利用できますか？">
        Benzinga は、多くの場合、新規ユーザーが有償プランを契約する前に API を試用できるトライアルまたはサンドボックス環境を提供しています。サンドボックス環境では、本番環境と比べてアクセスできるデータが限定されていたり、機能が一部制限されている場合があります。
    </Accordion>

    <Accordion title="Benzinga は API 連携に関するカスタマーサポートを提供していますか？">
        はい。Benzinga には、API の連携および利用を支援する専任のサポートチームがあります。加えて、Benzinga API のドキュメントには、ユーザーが API をアプリケーションにスムーズに統合できるよう、サンプルやコードスニペットも掲載されています。
    </Accordion>

    <Accordion title="ショートインタレストや政府取引など、プレミアムデータには追加料金がかかりますか？">
        ショートインタレストや政府取引などの特定のデータポイントには、プレミアムサブスクリプションや追加料金が必要となる場合があります。追加料金の詳細については、Benzinga のセールスチームに問い合わせるか、料金に関するドキュメントを確認してください。
    </Accordion>
</AccordionGroup>
