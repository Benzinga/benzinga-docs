title: "カンファレンスコール書き起こしストリーム"

---

<div id="websocket-websocketv1">
  ## ウェブソケット `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v1/earnings-call-transcripts/stream
```


<div id="parameters">
  ### パラメータ:
</div>

| Name       | Location |  Type   | Required | Description                                                             |
| :--------- | :-----:  | :-----: | -------: | ----------------------------------------------------------------------: |
| `token`    | query    | string  |      Yes | Benzinga のウェブソケット用トークン (bz.production***)。                 |
| `tickers`  | query    | string  | Optional | 購読するティッカーのリスト。カンマ区切りで指定します。                   |

<div id="stream-response">
  ### ストリームレスポンス:
</div>

ストリームにサブスクライブすると、カンファレンスコールのトランスクリプトに関する更新が、次の形式で配信されます。

```json
{
   "api_version":"EarningsCallTranscripts/v1",
   "kind":"stream_type",
   "data":{
      "type":"transcription_end",
      "content":{
         "call_id":"6757fbd6f0cc56000183c6a5",
         "transcript_chunk":"当社の第4四半期および2024年度通期の業績に非常に満足しています。",
         "security":{
            "company_name":"Limoneira",
            "ticker":"LMNR",
            "cik":"1342423",
            "isin":"US5327461043",
            "exchange":"NAS"
         },
         "time":1735189664,
         "index":0
      }
   }
}
```


<div id="responses">
  #### レスポンス
</div>

- **200 OK**
- **説明**: 成功
- **Content Type**: `application/json`
- **スキーマ**: [EarningsCallTranscriptsWSResp](#earningscalltranscriptswsresp)

<div id="earningscalltranscriptswsresp">
  ### EarningsCallTranscriptsWSResp
</div>

| Property      | Type   | Description                                                     |
|---------------|--------|-----------------------------------------------------------------|
| api_version   | string | ウェブソケット API のバージョン: `EarningsCallTranscripts/v1`。            |
| kind          | string | ウェブソケット接続ストリームの種別。                              |
| data          | object | ウェブソケットデータの詳細を含むオブジェクト。                    |
| - type      | string | メッセージ種別（例: transcript_chunk、transcription_end）。      |
| - content     | object | `EarningsCallTranscripts` スキーマへの参照。                         |

<div id="earningscalltranscripts">
  ### EarningsCallTranscripts
</div>

| Property           | Type    | Description                                                     |
|--------------------|---------|-----------------------------------------------------------------|
| call_id             | string  | 一意のコールID。    |
| transcript_chunk   | string  | カンファレンスコールの書き起こしチャンク。                            |
| security           | object  | インサイトが提供される銘柄（security）の詳細を含みます。  |
| - cik              | string  | 銘柄の CIK（Central Index Key）。                            |
| - exchange         | string  | 銘柄が上場している取引所。                        |
| - isin             | string  | ISIN（国際証券識別番号）。               |
| - name             | string  | 銘柄の名称。                                        |
| - symbol           | string  | 銘柄のティッカーシンボル。                               |
| time            | integer | トランスクリプトチャンクが作成された時刻のタイムスタンプ（float64形式）。 |
| index            | integer | トランスクリプトチャンクのインデックス（int形式）。 |
