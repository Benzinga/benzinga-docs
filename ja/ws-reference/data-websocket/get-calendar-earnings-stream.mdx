<div id="overview">
  ## 概要
</div>

リアルタイムの決算発表をサブスクライブし、1株当たり利益（EPS）、売上高、サプライズ要因などを含む企業業績の更新情報を即時に受信できます。

<div id="key-features">
  ### 主な機能
</div>

- **リアルタイム決算**: 決算発表が行われたタイミングで即時に通知を受け取ることができます
- **包括的なデータ**: EPS、売上高、予想値、実績値、サプライズ指標を含みます
- **銘柄でのフィルタリング**: 特定のティッカーまたは ISIN に購読できます
- **決算期間**: Q1、Q2、Q3、Q4 および通期（FY）の結果を追跡します
- **重要度レーティング**: 市場への重要度を 0〜5 段階のスケールで示します

<div id="use-cases">
  ### ユースケース
</div>

- リアルタイムの決算カレンダーとアラートを構築する
- 決算サプライズや予想超過／未達のパターンを追跡する
- ポートフォリオ内の銘柄の決算をリアルタイムでモニタリングする
- 決算シーズン向けのダッシュボードを作成する
- 決算結果に基づく売買戦略を自動化する

<div id="quick-start">
  ## クイックスタート
</div>

上部のインタラクティブなウェブソケットテスターを使って接続し、リアルタイムにストリームをテストしてください。

<div id="connection-url">
  ### 接続 URL
</div>

```
wss://api.benzinga.com/api/v2.1/calendar/earnings/stream?token=YOUR_TOKEN
```


<div id="query-parameters">
  ### クエリパラメータ
</div>

| Parameter | Required | Description |
|-----------|----------|-------------|
| `token` | Yes | Benzinga ウェブソケット API のトークン（bz.production***） |
| `tickers` | No | ティッカーシンボルのカンマ区切りリスト（例: `AAPL,MSFT`） |
| `isins` | No | フィルタに使用する ISIN（国際証券識別番号）のカンマ区切りリスト |

<div id="key-fields">
  ### 主要フィールド
</div>

| Field | 説明 |
|-------|-------------|
| `eps` | 報告された実際の1株当たり利益 |
| `eps_est` | アナリストコンセンサスによるEPS予想 |
| `eps_surprise` | 実績値と予想値の差分 |
| `eps_surprise_percent` | 予想値に対するサプライズの割合（％） |
| `revenue` | 報告された実際の売上高 |
| `revenue_est` | アナリストコンセンサスによる売上高予想 |
| `importance` | 市場における重要度（0〜5、5が最大） |

<div id="message-actions">
  ### メッセージアクション
</div>

| アクション | 説明 |
|--------|-------------|
| `created` / `Created` | 新しい決算発表が作成された |
| `updated` / `Updated` | 決算データが更新された |
| `deleted` / `Deleted` | 決算発表が削除された |

<div id="interactive-commands">
  ## インタラクティブコマンド
</div>

| Command | Description |
|---------|-------------|
| `ping` | 接続を維持します（`pong` で応答します） |
| `replay` | 最大 100 件までの直近のキャッシュ済みメッセージを再送します |

<div id="best-practices">
  ## ベストプラクティス
</div>

- **重複排除**: 重複処理を防ぐために `id` フィールドを使用する
- **重要度フィルター**: 市場インパクトの大きい決算では、重要度 4〜5 を重視する
- **サプライズアラート**: 大きな決算サプライズに対してアラートのしきい値を設定する
- **時刻**: `time` フィールドは、EST/EDT における24時間表記である点に注意する
