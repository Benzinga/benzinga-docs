title: "[Calendar] Earnings ストリーム"

<div id="websocket-websocketv1">
  ## ウェブソケット `websocket/v1`:
</div>

```txt
wss://api.benzinga.com/api/v2.1/calendar/earnings/stream
```


<div id="parameters">
  ### パラメータ
</div>

| Name       | Location |  Type   | Required | Description                                                                |
| :--------- | :-----:  | :-----: | -------: | -------------------------------------------------------------------------: |
| `token`    | query    | string  |      Yes | Benzinga ウェブソケット用のトークン (bz.production***)。                   |
| `tickers`  | query    | string  | Optional | 購読対象のティッカー一覧。カンマ区切り。                                   |
| `isins`    | query    | string  | Optional | フィルタに使用する ISIN（国際証券識別番号）の一覧。カンマ区切り。         |

<div id="responses">
  #### レスポンス
</div>

- **200 OK**
- **説明**: 成功
- **Content Type**: `application/json`
- **スキーマ**: [EarningsWSResp](#earningswsresp)

<div id="data-structures">
  ## データ構造
</div>

<div id="earningswsresp">
  ### EarningsWSResp
</div>

決算データ用の主なレスポンス構造体です。

| Parameter      | Type   | Description                                               |
|----------------|--------|-----------------------------------------------------------|
| id             | string | ウェブソケットレスポンスの一意なUUID                      |
| api_version    | string | ウェブソケットAPIのバージョン                            |
| kind           | string | ウェブソケット接続のストリームタイプ                     |
| data           | object | 決算データのアクションおよび詳細を含みます               |

<div id="data-object-inside-earningswsresp">
  ### データオブジェクト（EarningsWSResp 内）
</div>

| Parameter      | Type   | Description                                                   |
|----------------|--------|---------------------------------------------------------------|
| action         | string | ウェブソケットのデータアクション（例: created、updated、deleted） |
| id             | string | 決算データの ID                                               |
| content        | object | 決算の詳細（後述参照）                                       |
| timestamp      | string | データのタイムスタンプ                                       |

<div id="earnings-schema-inside-data-content">
  ### Earnings スキーマ（data content 内）
</div>

| Parameter               | Type     | Description                                               |
|-------------------------|----------|-----------------------------------------------------------|
| id                      | string   | 決算の一意な識別子                                         |
| date                    | string   | カレンダー上で発表された日付                               |
| date_confirmed          | string   | 決算発表日が確定しているかどうか（1/0）                    |
| time                    | string   | カレンダー上で発表された時刻（24時間制）                  |
| ticker                  | string   | ティッカーシンボル（F、MSFT など）                         |
| exchange                | string   | 取引所（NYSE、NASDAQ など）                               |
| name                    | string   | 証券の名称                                                 |
| currency                | string   | データの通貨建て                                           |
| period                  | string   | 決算実績の会計期間（Q1, Q2, Q3, Q4, FY, H1, H2）          |
| period_year             | integer  | 決算実績の会計期間の年                                     |
| eps_type                | string   | EPS の種類（Adj, GAAP, FFO）                              |
| eps                     | string   | 比較可能かつ調整後の 1 株当たり利益（EPS）                |
| eps_est                 | string   | 調整後 EPS のコンセンサス集計アナリスト予想              |
| eps_prior               | string   | 前会計期間の調整後 EPS                                    |
| eps_surprise            | string   | 予想値からの EPS の乖離                                   |
| eps_surprise_percent    | string   | 予想値からの乖離（パーセンテージ）                        |
| revenue_type            | string   | 売上高の種別（Adj, GAAP, FFO）                            |
| revenue                 | string   | 売上高                                                     |
| revenue_est             | string   | 売上高予想                                                 |
| revenue_prior           | string   | 前会計期間の売上高                                        |
| revenue_surprise        | string   | 予想値からの売上高の乖離                                  |
| revenue_surprise_percent | string   | 予想値からの乖離（パーセンテージ）                        |
| importance              | integer  | 本イベントが市場に与える重要度（0〜5）                    |
| notes                   | string   | Benzinga Newsdesk による補足メモ                          |
| updated                 | integer  | 最終更新日時のタイムスタンプ（UTC）                       |
